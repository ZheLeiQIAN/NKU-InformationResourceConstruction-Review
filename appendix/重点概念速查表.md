## C. 易混淆概念辨析表

| 概念A | 概念B | 核心区别 |
|-------|-------|----------|
| ISBN | ISSN | ISBN 13位用于图书；ISSN 8位用于连续出版物 |
| 图书 | 连续出版物 | 图书一次性出版，有ISBN；连续出版物定期/不定期出版，有ISSN |
| 霍顿定义 | 乌家培定义 | 霍顿用单复数区分；乌家培用狭义广义区分 |
| 价值论 | 需要论 | 价值论重文献质量轻读者需求；需要论重读者需求轻社会导向 |
| 文献增长 | 文献老化 | 增长看"产生"，老化看"失效"；一个向前看，一个向后看 |
| 会计成本 | 机会成本 | 会计成本是实际支出；机会成本是放弃的收益 |
| 稀缺型特色 | 完备型特色 | 稀缺型是独具特色的部分馆藏；完备型是系统性的学科/类型收藏 |
| 拥有 | 存取 | 拥有是所有权；存取是使用权 |
| 金色OA | 绿色OA | 金色OA=出版商开放出版(作者付费)；绿色OA=作者自存档(知识库存储) |
| 首次销售(纸质) | 不适用(电子) | 纸质书适用首次销售原则(可转售/出借)；电子书采用授权模式(不适用) |
| 著作权人身权 | 著作权财产权 | 人身权恒久(署名/修改/保护完整)；财产权50年(复制/发行/传播等12项) |
| Copyright | Copyleft | Copyright=保留权利；Copyleft=权利左传(要求衍生作品同样开放) |
| Copyright | CC许可 | Copyright=保留全部权利；CC=选择性地保留部分权利 |
| 智能图书馆 | 智慧图书馆 | 智能重技术自动化；智慧核心是人，以用户体验为中心 |
| SCI | SSCI | SCI=自然科学引文索引；SSCI=社会科学引文索引 |
| CSSCI | CSCD | CSSCI=南大核心(社科)；CSCD=中科院(自然科学，"中国的SCI") |
| 三线典藏制 | 二线典藏制 | 三线=一线+二线+三线(三线制)；二线=一线+二线(变体，无基本典藏) |
| 复选标准9种 | 复选方法9种 | 标准是"依据什么判断"(内容/书龄等)；方法是"怎么执行"(滞架/目录比较等) |
| 静态质量 | 动态质量 | 静态=数据准确性/完整性；动态=模型训练与应用效果 |
| 委托创作著作权 | 职务作品著作权 | 委托创作=合同约定归属；职务作品=法人或非法人单位享有 |
| 多重价格 | 捆绑价格 | 多重价格=按品种/时间/数量差异化定价；捆绑=打包销售(印本捆绑等) |
