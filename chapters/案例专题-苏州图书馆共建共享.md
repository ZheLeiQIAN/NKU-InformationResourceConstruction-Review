## 【核心概念】

### ★ 必考概念

| 概念 | 定义/内容 | 关键要点 |
|------|-----------|----------|
| **总分馆制** | 以总馆为核心，分馆为节点，统一管理的图书馆服务体系 | 苏州模式：统一型、紧密型集中管理 |
| **"苏州模式"** | 统一型、紧密型集中管理的总分馆制模式 | 从"契约型"到"政府主导"的演变 |
| **"网上借阅 社区投递"** | 2014年首创，读者通过APP下单，图书通过物流配送到社区投递点 | 2-3个工作日内送达 |
| **智能立体书库** | 国内首个大型智能化书库，占地3000㎡，4库区，700余万册藏书 | ASRS + WMS |
| **ASRS** | 自动化存取系统（Automated Storage and Retrieval System） | 智能书库核心硬件 |
| **WMS** | 书库管理系统（Warehouse Management System） | 智能书库核心软件 |
| **"苏州·书仓"** | 苏州市公共图书馆资源共享服务平台 | "一个平台、一套标准、一个体系" |
| **"五统一"** | 统一总书目库、读者证编号和图书条码、借阅规则、图书调配、文献编目 | 全域扩展的核心标准 |
| **LSP** | 图书馆服务平台（Library Services Platform），基于微服务架构的统一云服务平台 | 技术架构核心 |
| **协同采购** | 通过统一平台协调各馆采购，降低重复率 | 2019年重复率88.53% |
| **"你选书 我买单"** | 读者参与选书，图书馆负责采购的服务模式 | 读者驱动采购 |
| **"图书馆+"** | 与党群服务中心、学校、银行、商业中心、工业园等合作建设服务点 | 服务网络扩展模式 |
| **"e读书"小程序** | 2023年6月上线，整合30万种电子图书、1万种听书、3500种期刊 | 数字资源服务平台 |

### ☆ 重要概念

| 概念 | 定义/内容 | 关键要点 |
|------|-----------|----------|
| **契约型垂直管理模式** | 2005-2010年苏州总分馆制初期的管理模式，通过协议约束 | 全委托下的统一管理 |

| **网借中心平台** | 包括借阅管理、积分管理、物流管理、运营支撑、监控、数据中心等模块 | 六大模块 |

| **区块链存储** | 用于读者证信息存储，确保数据安全可信 | 新技术应用 |
| **12种书目判重规则** | 自动识别并合并重复条目的规则体系 | 书目统一技术 |
| **特征值计算和属性量化分析** | 对每本书目进行计算和分析，辅助完善书目数据 | 书目统一技术 |
| **统一借阅规则** | 成人10册、少儿10册，借阅时长30天，续借2次（每次10天），超期无法借阅 | 规则标准化 |
| **"1+7"个总馆** | 苏州图书馆+7个区县图书馆 | 联盟体系架构 |

## 【方法比较】

### 1. 苏州总分馆制三阶段管理模式比较 ★★★

| 维度 | 自发探索期（2005-2010） | 示范区建设期（2011-2013） | 全域扩展期（2020至今） |
|------|------------------------|--------------------------|------------------------|
| **管理模式** | 契约型垂直管理 | 政府主导 | 联盟协同 |
| **约束机制** | 协议约束 | 法规制度（实施方案） | 统一平台标准 |
| **建设主体** | 分馆建设主体（基层） | 市、区（县）两级政府 | "1+7"总馆联盟 |
| **经费来源** | 分馆建设主体承担 | 政府财政保障 | 统一协同采购 |
| **分馆数量** | 26个（2010年） | 182个（2013年） | 477个（2024年） |
| **核心特征** | 全委托下的统一管理 | 统一型、紧密型集中管理 | "五统一"标准体系 |
| **主要问题** | 主体多头、缺乏法规、布局不科学、难持续 | - | - |
| **标志性成果** | 2009年"第十四届群星（服务）奖" | 首批国家公共文化服务体系示范区 | 国家智慧图书馆创新应用优秀案例 |

**演变逻辑**：契约型（协议约束）→ 政府主导（制度保障）→ 联盟协同（平台统一）

### 2. "网上借阅 社区投递"三种配送方式

| 配送方式 | 操作流程 | 适用场景 |
|----------|----------|----------|
| **就近投递** | 图书配送至读者指定的社区投递点 | 读者自取 |
| **送书上门** | 物流直接配送至读者指定地址 | 行动不便/特殊需求 |
| **柜上借还** | 读者在智能书柜上自助借还 | 轨交站、商业中心等人流密集区 |

### 题目2（名词解释）

**解释"苏州模式"和"五统一"的内涵。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

**"苏州模式"**：

苏州图书馆在总分馆制建设中形成的"统一型、紧密型集中管理"模式。其核心特征包括：
- **统一型**：统一采编、统一调配、统一管理、统一服务标准
- **紧密型**：总馆与分馆之间建立紧密的业务联系和资源共享机制
- **集中管理**：通过政府主导，将分馆建设主体上移到市、区（县）两级政府，实现人财物的统筹配置

该模式经历了从"契约型垂直管理"（2005-2010）到"政府主导"（2011-2013）再到"联盟协同"（2020至今）的演进。

**"五统一"**：

苏州市公共图书馆联盟在全域扩展期实施的五大统一标准：
1. **统一总书目库**：从各成员馆抽取开放书目数据，依据12种书目判重规则自动识别并合并重复条目
2. **统一读者证编号和图书条码**：线上实人认证，系统自动检查，采用区块链技术存储
3. **统一借阅规则**：成人10册、少儿10册，借阅时长30天，续借2次（每次10天），超期无法借阅
4. **统一图书调配**：建设一级中心（苏州图书馆人民路馆）和二级中心（各成员馆物流中心），建立图书临时中转库和分拣线
5. **统一文献编目**：统一手工编目工具、馆配商提供编目数据导入、集成标准编目数据套录

**解析**：本题是高频名词解释。"苏州模式"需强调"统一型、紧密型集中管理"八个字；"五统一"需完整列出五项内容。注意"五统一"是2020年后全域扩展期的核心标准，与早期的"契约型管理"有本质区别。

### 题目5（案例分析题）

**苏州图书馆在2020年启动资源共享平台建设时发现"各区县图书馆总分馆各自独立，人财物管理各自为政，馆藏资源无法共享，业务系统各异"。请分析这一问题的成因，并说明苏州图书馆如何通过技术手段和管理创新解决这一问题。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

**一、问题成因分析**

1. **体制因素**：各区县图书馆分属不同行政层级和财政体系，缺乏统一协调机制
2. **技术因素**：各馆采用不同的业务系统，数据格式不统一，无法互联互通
3. **标准因素**：缺乏统一的书目标准、编目规则、借阅规则
4. **管理因素**：人财物管理各自为政，采购重复（2019年重复率达88.53%）
5. **历史因素**：早期建设缺乏顶层设计和长远规划

**二、技术手段**

1. **统一技术平台**
   - 采用LSP（图书馆服务平台）构建基于微服务架构的统一云服务平台
   - 通过API接口实现与中心平台对接和数据交互
   - 通过ETL实现各馆书目数据、借阅数据、读者数据的抽取清洗和转换
   - 通过大数据分析平台实现整合型数据的规模化分析和智能决策

2. **书目统一技术**
   - 从各成员馆抽取开放书目数据
   - 依据12种书目判重规则自动识别并合并重复条目
   - 对每本书目进行特征值计算和属性量化分析
   - 利用400多万标准书目和图书封面库辅助完善

3. **数据安全技术**
   - 线上实人认证，系统自动检查读者证号码
   - 采用区块链技术存储信息
   - RFID全兼容支持所有成员馆

4. **智能决策技术**
   - 利用人工智能建模技术分析数据，建立资源利用率预测模型
   - 预测采购后未来一年流通率
   - 基于大数据分析建立文献资源预警与调拨机制

**三、管理创新**

1. **标准统一："五统一"**
   - 统一总书目库、读者证编号和图书条码、借阅规则、图书调配、文献编目
   - 统一借阅规则：成人10册、少儿10册，30天，续借2次（每次10天）

2. **组织创新：联盟机制**
   - 2021年1月成立苏州市公共图书馆联盟，8个图书馆签订框架协议
   - 从"各自为政"到"协同合作"

3. **采购创新：协同采购**
   - 构建"协同待采目录"
   - "你选书 我买单"服务
   - 预计每年多采购6万种图书，降低88.53%的重复率

4. **物流创新：分级中心**
   - 一级中心（苏州图书馆人民路馆）+ 二级中心（各成员馆物流中心）
   - 建立图书临时中转库和分拣线
   - 调配响应时间从3天压缩至1天

**四、成效**

- 2022年底实现市域范围纸质图书通借通还，可借阅图书600万册
- 2023年完成跨馆调配图书45万册，缺书率下降65%，运输成本降低30%
- 2024年联盟体系达"1+7"总馆、477分馆、322共享取书点

**解析**：本题是案例分析题。答题时需先分析问题成因（多角度），再分别阐述技术手段和管理创新，最后以数据验证成效。注意"五统一"和协同采购是核心创新点，需重点展开。
