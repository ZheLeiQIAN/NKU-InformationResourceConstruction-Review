## 【核心概念】

### ★ 必考概念

| 概念 | 定义 | 关键要点 |
|------|------|----------|
| **馆藏文献布局** | 将图书馆入藏的全部文献，按一定标准划分为相对独立而又相互联系的若干部分，建立各种功能的书库、阅览空间，为每部分藏书确定合理存放位置 | 实质是空间位置上的划分 |
| **三线典藏制（Three-Tier Stacking System）** | 按信息资源新旧程度及利用率高低，结合服务方式，将馆藏划分为利用率最高、较高、低三部分，依次组成一、二、三线书库的布局方法 | 依据特鲁斯威尔"二八率"；一线50-60%/二线20-30%/三线10% |
| **二八率（Trueswell's 80/20 Rule）** | 美国图书馆学家特鲁斯威尔提出：馆藏中约20%是常用的，能满足读者80%的需求；其余80%仅满足20%的需求 | 三线典藏制的理论基础 |
| **复选（Re-selection）** | 在藏书补充过程后一阶段，对馆内入藏书刊的再次选择；与初选（馆外出版物入藏选择）构成辩证统一过程 | 苏联格利科尔耶夫60年代提出，斯多利亚洛夫1979系统阐述 |
| **剔除（Weeding/Deselection）** | 根据一定原则和标准，将长期滞留在书架上、利用率极低或已失去利用价值的文献从馆藏文献体系中分离出来，并按不同情况分别处理 | 复选的结果性操作 |
| **半衰期（Half-life）** | 某学科被利用的文献总量中，一半文献失去利用效率所经历的时间；1960年伯顿（R.Burton）和凯普勒（R.Kebler）提出 | 从文献年龄方面为藏书剔除提供依据 |
| **滞架时限（Shelf-stand Time）** | 文献在两次流通之间（含最后一次流通至今）滞留在书架上未被使用的时间长度 | 确定流通书库中文献馆内调配和去留的**最佳标准** |
| **贮存图书馆（Depository Library）** | 又称保存图书馆、储备图书馆、储存图书馆、寄存图书馆，为解决收藏空间紧张、节省管理成本、提高利用率而设立的专门收藏利用率相对较低文献资源的图书馆 | 5种模式 |
| **COUNTER标准** | Counting Online Usage Network Electronic Resources，由图书馆界、出版界和中间商共同发起，2002年3月正式启动，规范网络化电子资源使用统计的国际化标准 | 起源于英国，现已成为国际标准 |

### ☆ 重要概念

| 概念 | 定义 | 关键要点 |
|------|------|----------|
| **馆藏文献排列（排架）** | 将馆藏文献有序列地陈放在书架上，形成一定检索系统，使每种文献都有固定位置 | 内容排架法+形式排架法 |
| **分类排架法** | 按文献本身内容所属的学科体系来排列藏书的方法；由分类号和辅助号组成分类排架号 | **主要的馆藏文献排列方法** |
| **辅助号** | 分类排架法中区分同类图书的号码，通常有著者号（字顺号）、种次号、登记号等 | 分类号相同再按辅助号排列 |
| **专题排架法** | 将出版物在一定专题范围集中起来，向读者宣传推荐，带有专架陈列、专架展览性质 | 横向范围集中，打破学科界线，临时性排架 |
| **固定排架法** | 按出版物的固定编号排架；编号由四组号码组成：库室号、书架号、层格号、书位号 | 记住四组号码 |
| **二线典藏制** | 三线典藏制的变体，取消三线制的二线书库即成为二线典藏制 | 适合中小型图书馆 |
| **初选（Initial Selection）** | 补充过程开始阶段对馆外出版物的入藏选择 | 多通过征订目录间接挑选，不可避免出现失误 |
| **保障标准** | 以文献的保障要求作为复选标准；馆际协作计划确定由本馆保障的文献不进行复选或仅剔除多余复本 | 9种复选标准之一 |
| **馆藏文献信息处置** | 公共图书馆根据工作需要，对纳入正式馆藏的文献信息进行产权转让或注销产权的行为 | 《公共图书馆馆藏文献信息处置管理办法》2022年4月7日印发 |
| **布拉德福定律（Bradford's Law）** | 学科文献在期刊中的分布呈集中与离散规律，可用于测定检索工具完整性 | 论文搜全率=实测值/理论值 |
| **专家评价法** | 由有学科专业背景的学者、专家、馆员、顾问或图书馆认证委员会根据图书馆方针目的，直接观察馆藏进行评价 | 简便易行 |
| **引文分析法** | 通过查对学术性论著的注释和参考文献，检查作者引用的参考文献被本馆收藏的情况 | 反映馆藏被利用和可能被利用的情况 |

## 【重点公式/算法】

> 本章涉及布局比例、排架号码结构、统计公式等，以下为需记忆的关键数字与结构。

### 1. 三线典藏制比例结构 ★★★

```
全部馆藏
│
├─ 一线书库（开架辅助书库）   利用率最高 50-60%   最新出版文献
├─ 二线书库（闭架/限制开架）  利用率较高 20-30%   近期出版文献
└─ 三线书库（基本典藏书库）   利用率低   10%      罕用/过时/备查文献
```

**记忆口诀**：一线五六最新书，二线二三近期刊，三线一成罕用本。

**二八率依据**：20%馆藏满足80%需求，80%馆藏满足20%需求。

### 2. 分类排架号结构

```
分类排架号 = 分类号 + 辅助号
             │          │
             │          └─ 区分同类图书：著者号 / 种次号 / 登记号
             └─ 代表图书内容所属的学科类目
```

**常用组合**：分类著者号排架、分类种次号排架、分类书名号排架。

### 3. 固定排架号结构

```
固定排架号 = 库室号 + 书架号 + 层格号 + 书位号
```

### 4. 馆藏拒借率公式

```
                    读者未借到馆藏的数量
馆藏拒借率 = ─────────────────────────── × 100%
                    读者要求借阅的总数量
```

### 5. 种（册）利用率公式

```
                 读者利用的馆藏种（册）数
种（册）利用率 = ────────────────────── × 100%
                 馆藏总种（册）数
```

### 6. 布拉德福定律检索工具完整性测定公式

```
曲线部分：R(n) = αn^β           （1 ≤ n < c）
直线部分：R(n) = N·ln(n/s)      （c ≤ n ≤ N）

当N足够大时，K ≈ N

论文搜全率 = 实测值R(n) / 理论值R(n) × 100%

实例（CNKI"数字图书馆"主题）：
  理论值 R(n) = 2186
  实测值 R(n) = 1775
  论文搜全率 = 1775/2186 × 100% = 81.2%
```

### 7. 复选标准9种速记

```
内容 · 外形 · 书龄       ← 文献本体属性
滞架时限 · 使用频率       ← 使用状况指标
复本量 · 替代 · 语种     ← 馆藏配置指标
保障                      ← 协作与体系要求
```

**记忆口诀**：内外形书龄，滞架频率行，复本替语种，保障九标准。

## 【方法比较】

### 1. 三线典藏制 vs 二线典藏制

| 比较维度 | 三线典藏制 | 二线典藏制 |
|----------|-----------|-----------|
| **书库层数** | 一线、二线、三线书库 | 一线、二线书库（取消原三线制的二线） |
| **适用馆型** | 藏书较多、馆舍较大的大型或超大型图书馆 | 中小型图书馆 |
| **关系** | 原型 | 三线典藏制的变体 |
| **核心依据** | 特鲁斯威尔"二八率" | 同三线典藏制 |

### 2. 内容排架法 vs 形式排架法

| 比较维度 | 内容排架法 | 形式排架法 |
|----------|-----------|-----------|
| **排列依据** | 文献内容所属学科体系 | 文献的外形特征或编号顺序 |
| **包含方法** | 分类排架法、专题排架法 | 登记号、固定、字顺、年代、文种、书型、色标排架法 |
| **优点** | 内容相同相近的书集中，便于按类检索；便于馆员熟悉研究藏书 | 号码单一位置固定，节省空间，不产生倒架 |
| **缺点** | 不能充分利用空间；倒架耗费人力；排架号长速度慢易出错 | 同类同复本书不能集中，不便在书架上检索研究 |
| **地位** | 分类排架法是**主要**的排列方法 | 书型排架法等为**辅助性**方法 |

### 3. 分类排架法 vs 专题排架法

| 比较维度 | 分类排架法 | 专题排架法 |
|----------|-----------|-----------|
| **集中方式** | 按学科体系纵向集中 | 按专题范围横向集中 |
| **学科界线** | 严格遵循学科隶属关系 | 打破学科隶属界线 |
| **性质** | 常规性、长期性排架 | 临时性排架（宣传推荐性质） |
| **使用场景** | 全馆主要排架方式 | 外借处、阅览室、开架书库宣传专题新书 |

### 4. 形式排架法7种比较

| 排架法 | 排列依据 | 特点 |
|--------|---------|------|
| **登记号排架法** | 个别登记顺序 | 一书一号，简单清楚，不反映内容 |
| **固定排架法** | 固定编号（库室号+书架号+层格号+书位号） | 位置固定，不倒架 |
| **字顺排架法** | 书名或著者名称字顺 | 号码单一易记，同类同复本不集中 |
| **年代排架法** | 出版年代顺序 | 按时间排列 |
| **文种排架法** | 语言文别 | 适用于外文书刊 |
| **书型排架法** | 外形特征/规格/装帧 | 辅助性方法，与分类排架结合成分类字顺排架法 |
| **色标排架法** | 颜色标识 | 辅助性视觉标识 |

### 5. 复选标准9种比较

| 标准 | 核心判断依据 | 关键要点 |
|------|------------|----------|
| **内容标准** | 文献内容是否陈旧过时、重复、不符合馆藏政策 | 包括内容不适于公开流通 |
| **外形标准** | 文献外观是否陈旧、污损、缺页、破损 | 视听/缩微/电子文献含制作质量与读取能力 |
| **书龄标准** | 出版/版权/印刷/采购/进入流通日期至复选时的年限 | 以时间年限衡量 |
| **滞架时限标准** | 两次流通间滞留书架未被使用的时间长度 | **最佳标准**（确定流通书库文献去留） |
| **使用频率标准** | 一定时期内流通、阅览、检索或下载次数 | 低于规定次数为复选对象 |
| **复本量标准** | 复本量多寡是否符合读者需求 | 多余复本+复本不足两种情况 |
| **替代标准** | 不同类型文献间的相互替代性 | 主要考虑电子文献对印刷型文献的替代 |
| **语种标准** | 馆藏文献的语种 | 语种与读者掌握语种不一致、无人问津的语种 |
| **保障标准** | 文献的保障要求 | 馆际协作/馆藏体系明确要求保障的不复选或仅剔复本 |

### 6. 复选方法9种比较

| 方法 | 操作原理 | 优缺点/前提条件 |
|------|---------|----------------|
| **经验判断法** | 根据文献外观、价值、读者取向、借阅频率等感性认识和经验决定取舍 | 简单但缺乏精确，主观差异大 |
| **滞架时限法** | 根据文献两次流通间滞留未被使用的时间长度确定去留 | 确定"呆滞书刊"的有效方法，前提是完整借阅记录 |
| **书龄法** | 根据书龄长短确定取舍 | 依据半衰期理论，分析不同书龄馆藏结构和使用状况 |
| **半衰期测定法** | 根据学科文献半衰期规律确定各类藏书取舍 | 需先确定各学科文献不同有效时间 |
| **目录比较法** | 将馆藏目录与推荐书目、权威书目比较进行复选 | 如比较馆藏期刊目录与核心期刊目录 |
| **用户评议法** | 通过访问、交谈或表格调查了解用户评价与建议 | 反映读者需求，但受用户主观影响 |
| **内容层次判断法** | 将文献按内容深浅分层次，低于所需层次的列为剔除对象 | 适用于层次分明的馆藏体系 |
| **数学计算方法** | 用数量表示变量，运用公式计算后做出复选决定 | 载文量、引文量、书龄、复本量、滞架时限、使用频率为重要参数 |
| **外形判断方法** | 以文献外形为剔除标准 | 多次修补无法使用、质量差影响阅读、污损缺页严重者剔除 |

### 7. 贮存图书馆5种模式比较

| 模式 | 建立主体 | 服务对象 |
|------|---------|---------|
| **专有贮存图书馆** | 各图书馆自行建立 | 本馆利用率较低的书刊资料 |
| **地区合作贮存图书馆** | 一定地区内若干图书馆联合建立 | 本地区各参加馆剔除的罕用文献 |

| **国际性组织出版物贮存图书馆** | 国际性组织指定 | 该组织出版物的特定贮藏 |
| **特种文献贮存图书馆** | 多个图书馆共同建立 | 某一特定类型文献（如缩微品、未出版资料） |

### 题目1（名词解释）

**请解释"三线典藏制"的概念、理论依据及各线书库特征。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

**三线典藏制**是按照信息资源的新旧程度及利用率的高低，结合服务方式方法，将全部馆藏信息资源划分为利用率最高的、比较高的和利用率低的三部分，并依次组成一、二、三线书库的布局方法。

**理论依据**：美国图书馆学家特鲁斯威尔（Trueswell）经过大量统计分析，总结出馆藏信息资源利用的"二八率"规律——在图书馆全部馆藏中，大约有20%是常用的，通常能满足读者80%的需求；其余80%的馆藏仅能满足读者20%的需求。

**三线书库特征**：
- **一线书库**：开架辅助书库，利用率最高（50-60%），针对性最强，专门存放最新出版的文献资料
- **二线书库**：闭架或限制开架的辅助书库，利用率较高（20-30%），参考性较强，专门存放近期出版的文献资料
- **三线书库**：基本典藏书库，利用率低（10%），主要存放各种罕用、过时失效及内部备查等文献资料

**解析**：本题是高频名词解释与简答题。需准确记忆"二八率"的提出者、20%/80%的对应关系，以及三线书库各自的利用率比例（50-60%/20-30%/10%）和藏书特征。注意区分"开架辅助书库""闭架或限制开架""基本典藏书库"三种书库类型。

</details>

### 题目3（简答题）

**简述馆藏文献复选与剔除的9种基本标准。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

馆藏文献复选与剔除的基本标准共9种：

1. **内容标准**：以文献内容是否陈旧过时、重复、不符合馆藏政策或不适于公开流通为标准
2. **外形标准**：以文献外观是否陈旧妨碍使用、纸张印刷装订质量、污损缺页、多次修补无法使用等为标准；视听/缩微/电子文献还包括制作质量、读取能力、老化变质等
3. **书龄标准**：以文献的出版日期、版权日期、印刷日期、采购日期或进入流通日期至复选时所经历的年限为标准
4. **滞架时限标准**：以文献在两次流通之间滞留在书架上未被使用的时间长度为标准，是确定流通书库中文献馆内调配和去留的**最佳标准**
5. **使用频率标准**：以一定时期内的流通、阅览、检索或下载次数为标准，少于规定次数的为复选对象
6. **复本量标准**：以文献复本量多寡是否符合读者需求为标准，包括多余复本和复本不足两种情况
7. **替代标准**：以不同类型文献间的相互替代性为标准，主要考虑电子文献对印刷型文献的替代性
8. **语种标准**：以馆藏文献的语种为标准，包括语种与读者掌握语种不一致、无人问津的语种、译自中文文献的不常用语种版本
9. **保障标准**：以文献的保障要求为标准，馆际协作计划确定的由本馆保障的文献不进行复选或仅剔除多余复本

**解析**：本题是高频简答题。关键在于完整列举9种标准，并指出滞架时限标准是"最佳标准"。记忆口诀："内外形书龄，滞架频率行，复本替语种，保障九标准。"

</details>

### 题目4（简答题）

**简述馆藏文献复选的9种方法。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

馆藏文献复选方法共9种：

1. **经验判断法**：根据文献外观、价值、读者利用取向、复本需求、借阅频率等的感性认识和平时经验决定取舍；相对简单但缺乏精确
2. **滞架时限法**：根据文献在两次流通之间滞留未被使用的时间长度确定去留；是确定"呆滞书刊"的有效方法，前提是有完整借阅记录
3. **书龄法**：根据书龄长短确定取舍；依据半衰期理论，通过分析不同书龄馆藏结构和使用状况确定重点复选年代范围
4. **半衰期测定法**：根据不同学科文献的半衰期规律确定各类藏书取舍；需先确定各学科文献不同有效时间
5. **目录比较法**：将馆藏目录与推荐书目、权威书目比较进行复选；如比较馆藏期刊目录与核心期刊目录
6. **用户评议法**：通过访问、交谈或表格调查等用户调查方式了解用户对文献资源建设的评价与建议
7. **内容层次判断法**：将文献按内容深浅分为几个层次，归入相应层次，低于所需层次的列为剔除对象
8. **数学计算方法**：用数量表示某些变量并运用专门公式计算，根据计算结果做出复选决定；载文量、引文量、书龄、复本量、滞架时限和使用频率为重要参数
9. **外形判断方法**：以文献外形为剔除标准；多次修补无法使用、纸张印刷装订质量差严重影响阅读、污损缺页严重的属于剔除范围

**解析**：本题考查复选方法的完整性。注意与复选标准（9种）区分——标准是判断依据，方法是操作手段。记忆口诀："经验滞架书龄，半衰目录用户，内容数学外形。"

</details>

### 题目5（简答题）

**简述馆藏信息资源评价的6种方法。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

馆藏信息资源评价方法共6种：

1. **专家评价法**：由有学科专业背景的学者、专家、馆员、顾问或图书馆认证委员会根据图书馆方针目的，直接观察馆藏进行评价。简便易行，可了解馆藏范围、规模、深度等大致情况。

2. **用户评议法**：以口头访谈、书面问卷和网络调查形式向用户征求对馆藏的意见，了解读者需求情况和馆藏利用效能情况。

3. **统计分析法**：运用各种馆藏统计数据进行分析和评价，是运用广泛且有效的方法。包括馆藏数量统计、流通阅览统计（藏书流通率、拒借率、种册利用率）、检索利用统计。数字资源使用统计可参考COUNTER标准。

4. **书目核对法**：将被评价馆藏与标准书目、核心书目或权威性馆藏目录逐一核对以评价馆藏文献质量。注意书目应由学科专家选择、及时更新，且不同图书馆使用同一书目不一定恰当。

5. **引文分析法**：通过查对学术性论著的注释和参考文献，检查引用文献被本馆收藏的情况，以及馆藏被利用和可能被利用的情况。

6. **成本收益分析法**：对数字资源的成本与收益进行比较以评价馆藏质量及经费分配合理性。成本包括7项（资源购置和自建成本、设备成本、存储成本、软件成本、维护和管理成本、处置和保存成本、参与联盟的成本）；收益包括用户信息获取保障情况和用户成果产出效益。

**解析**：本题是高频简答题。需完整列举6种方法并简述各自原理。记忆口诀："专家用户统计，书目引文成本。"注意统计分析法中包含COUNTER标准这一子知识点。

</details>

### 题目8（论述题）

**论述三线典藏制的理论依据、各线书库特征及其对馆藏管理的意义。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

**一、理论依据**

三线典藏制依据美国图书馆学家特鲁斯威尔（Trueswell）提出的"二八率"规律：在图书馆全部馆藏信息资源中，大约有20%是常用的，通常能满足读者80%的需求；其余80%的馆藏信息资源仅能满足读者20%的需求。根据这一理论，国外图书馆普遍采用三线典藏制来布局馆藏信息资源。

**二、各线书库特征**

| 书库 | 类型 | 利用率 | 藏书特征 |
|------|------|--------|---------|
| 一线书库 | 开架辅助书库 | 50-60% | 利用率最高，针对性最强，专门存放最新出版文献 |
| 二线书库 | 闭架/限制开架辅助书库 | 20-30% | 利用率较高，参考性较强，专门存放近期出版文献 |
| 三线书库 | 基本典藏书库 | 10% | 利用率低，存放罕用、过时失效及内部备查文献 |

**三、对馆藏管理的意义**

1. **改革传统布局模式**：改革了以基本书库为中心、以辅助书库为分支的传统藏书布局模式，将原来的辅助书库作为馆藏信息资源布局的主体，实现了藏阅一体化，提高了馆藏信息资源利用率
2. **建立动态联系**：一、二、三线书库形成了增加新书与剔除滞书的动态性联系，使馆藏保持活力
3. **适用范围明确**：三线典藏制适用于同一层次内藏书的横向布局
4. **变体应用**：对于中小型图书馆，可采用二线典藏制（取消三线制的二线书库），灵活适应不同规模图书馆的需求

**四、确定藏书利用率的方法**

- 依据一定的时限确定文献利用率
- 根据借阅记录统计分析文献利用率
- 根据经验判断文献利用率

**解析**：本题是综合性论述题。需从理论依据、各线特征、管理意义三个层面系统阐述。注意准确记忆三线比例数字（50-60%/20-30%/10%），并理解二八率中20%/80%的对应关系。论述时应体现三线典藏制从理论到实践、从布局到管理的完整逻辑。

</details>

### 题目9（简答题）

**简述《公共图书馆馆藏文献信息处置管理办法》的主要内容及其8种处置情形。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

《公共图书馆馆藏文献信息处置管理办法》由文化和旅游部于2022年4月7日印发，适用于中华人民共和国范围内由政府设立的公共图书馆进行馆藏文献信息处置。

**一、处置定义**：公共图书馆根据工作需要，对纳入正式馆藏的文献信息进行产权转让或注销产权的行为。

**二、处置原则**：
- 应当遵循合法合规、分级分类、科学规范、协同合作的原则
- 对需要长期保存的馆藏文献信息，确需处置的，处置时应当遵循**保留品种、剔除复本**的原则
- 馆藏文献信息属于文物、古籍、档案或者涉及国家秘密的，原则上不进行处置
- 处置方式包括无偿划转、捐赠、置换、转让、报损或报废等，优先选择有助于提高利用率的无偿划转、捐赠和置换

**三、8种处置情形**：
1. 损坏严重或者存储格式过时的
2. 复本量过多的
3. 流通率较低的
4. 内容过时的
5. 涉及盘亏以及非正常损失的
6. 因自然灾害等不可抗力造成毁损、灭失的
7. 因馆藏发展策略调整而需要进行处置的
8. 其他原因需要处置的

**解析**：本题是政策法规类高频考点。关键是完整列举8种处置情形，并理解"保留品种、剔除复本"这一核心原则。注意该办法与复选、剔除知识点的关联——处置是复选和剔除的制度化、规范化体现。

</details>

### 题目10（论述题）

**论述馆藏信息资源评价的标准与指标体系（4个维度），并说明各维度的评价方法。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

馆藏信息资源评价的标准与指标体系包括4个维度：

**一、对馆藏信息资源保障能力的评价**

评价图书馆拥有和可存取的馆藏数量是否满足读者需求。

- **（一）拥有的馆藏数量**：①印刷型馆藏评价标准主要有藏书保障率、读者满足率、对各学科文献的覆盖率等；②电子文献评估从信息类型、信息含量、利用能力和程度等方面衡量保障能力和对印刷型文献的替代程度
- **（二）可存取的馆藏数量**：①从网络上通过协议能连续稳定获取的专业数据库信息资源（虚拟馆藏）数量；②通过馆际互借、文献传递获取的信息资源数量；③通过开放存取、学科信息门户获取的免费资源数量

**二、对馆藏信息资源质量的评价**

- **（一）馆藏结构质量**：包括学科结构、等级结构、文种结构、时间结构、文献类型结构5种
- **（二）馆藏内容质量**：即馆藏的信息知识含量
- **（三）馆藏组织质量**：包括信息选择质量、信息组织水平、信息加工能力3项
- **（四）检索系统的功能**：包括检索途径和检索字段、检准率与检全率、用户体验3项

**三、对馆藏信息资源共享性的评价**

- **（一）馆藏文献数字化的可传播范围**
- **（二）特色数据库建设规模、效益**

**四、对馆藏信息资源利用情况的评价**

- **（一）馆藏利用率**：①印刷型文献通过外借率、利用率、拒借率、周转率、馆际互借满足率衡量；②数字资源通过单位时间内访问量、检索量、成功量、被拒量、失败量衡量
- **（二）读者满意度**：侧重读者感受，从用户使用期望和使用感知角度实施评价

**评价方法**：6种方法（专家评价法、用户评议法、统计分析法、书目核对法、引文分析法、成本收益分析法）可综合运用于上述4个维度的评价。其中统计分析法运用最广泛，可覆盖保障能力（数量统计）、利用情况（流通阅览统计、检索利用统计）等维度；成本收益分析法侧重于数字资源评价。

**解析**：本题是综合性论述题。需系统呈现4个评价维度及其子项，并理解4维度之间的逻辑关系（保障能力→资源质量→共享性→利用情况是一个从"有没有"到"好不好"到"能否共享"到"用没用"的递进逻辑）。

</details>
