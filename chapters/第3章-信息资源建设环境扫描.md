## 【核心概念】

### ★ 必考概念

| 概念 | 定义 | 关键要点 |
|------|------|----------|
| **开放获取（Open Access, OA）** | 科研知识成果在互联网公开发布，用户可免费阅读、下载、复制、保存、传播、使用，著作权人保留著作权 | 保留著作权+免费使用，区别于公有领域 |
| **金色OA（Gold OA）** | 开放出版（Open Access Publishing），通过发表在OA期刊上将研究成果开放获取 | 两种形式：开放期刊+开放论文；可能有APC |
| **绿色OA（Green OA）** | 开放存储（Open Access Archives），将研究成果存储到机构或学科知识库 | 自存储；SHERPA/RoMEO确定存档版本和时滞 |
| **首次销售原则（First Sale Doctrine）** | 著作权作品首次销售后，合法复制品所有人可自由处分（出借、转售等） | 纸质书适用，电子书不适用 |
| **公共借阅权（Public Lending Right, PLR）** | 版权人和邻接权人因其版权作品在图书馆被免费出借而获得报酬的权利 | 以英国为代表 |
| **Copyleft（著佐权）** | 由斯托曼最早提出，利用现有著作权体制保护所有用户和二次开发者自由的授权方式 | 仍在Copyright框架中；GNU/GitHub |
| **CC协议（Creative Commons）** | 知识共享，在传统"版权所有"外提出"部分权利保留"的灵活著作权行使模式 | 4核心元素by/nc/nd/sa，6组合 |
| **合理使用** | 在特定条件下，不经著作权人许可、不向其支付报酬而使用作品 | 图书馆数字资源建设面临3条障碍 |
| **第三空间（Third Place）** | 由Ray Oldenburg提出，指居住（第一空间）、工作（第二空间）之外的购物休闲空间 | 图书馆定位为第三空间 |
| **读者决策采购（PDA/DDA）** | 将读者阅读需求量化成指标作为文献采购决策依据的资源建设新模式 | 也称POD（按需购买）、DDA（需求驱动采购） |
| **智慧图书馆** | 基于微服务架构的统一云服务开放平台，以数据共享为纽带连接各功能模块 | "智能"重技术，"智慧"重人本 |
| **著作权财产权（12项）** | 复制权、发行权、出租权、展览权、表演权、放映权、广播权、信息网络传播权、摄制权、改编权、翻译权、汇编权 | 保护期一般为50年 |

### ☆ 重要概念

| 概念 | 定义 | 关键要点 |
|------|------|----------|
| **兰卡斯特"无纸化社会"预言** | 70年代末提出，纸印方式终将部分或全部被电子手段取代，21世纪将进入无纸化信息社会 | Frederick Wilfrid Lancaster |
| **OA2020** | 德国马普学会等机构发起，旨在将现有学术期刊从订购模式转换为开放出版模式 | 2016年3月21日正式发布 |
| **Plan S** | 11个欧洲国家13个资助机构联合发布的开放获取计划，含十项原则 | 2020年1月1日后公共资金资助成果须OA发布 |
| **APC（论文处理费）** | 开放获取期刊向作者收取的费用，以弥补开放后用户订购费用的缺失 | 由资助机构/大学支付，非研究人员 |
| **DOAJ** | 开放获取期刊目录，由瑞典Lund大学图书馆创建和维护 | 最有影响的OA期刊综合节点 |
| **预印本（Preprint）** | 科研成果未在正式刊物发表前，出于同行交流目的自愿传播的科研论文 | 比印刷版早1-2年；arXiv为代表 |
| **SHERPA/RoMEO** | 出版商自归档政策列表数据库，用于确定存档版本和开放时滞 | 绿色OA工具 |
| **期刊危机（Serials Crisis）** | 连续出版物支出增长远超通胀和图书馆预算，导致图书馆无力负担 | 支出占比从1998年25%增至2019年近40% |
| **主题图书馆** | 围绕某一特定主题组织馆藏资源并开展多样化服务的新型图书馆 | 杭州20+主题分馆；傅雷图书馆 |
| **Borgman数字图书馆定义** | 两层含义：电子资源与技术的集合；由用户社区构建的机构 | 数字图书馆是实体图书馆在网络环境的延伸 |
| **米老鼠保护法案** | 1998年"桑尼·波诺版权期限延长法案"，将有效作品版权期限延长20年 | 米老鼠版权延长到2024年 |
| **指南性数据库** | 包括馆藏书目数据库、联合书目数据库、特色文献数据库 | 图书馆作为生产者享自主知识产权 |
| **源数据库** | 读者可直接获取原始文献及有关数据的数据库 | 通常具有独立的著作权 |

### 2. 柏林宣言开放获取两个条件

```
条件一：作者/版权所有人承诺向所有用户提供
        ┌─ 免费的
        ├─ 不能撤回的
        └─ 全世界范围内的
        复制、利用、传播权利（须以适当方式表明作者权利）

条件二：作品的完整版本
        ┌─ 以标准格式存储
        └─ 存储到在线存储库中
        支持开放获取和长期保存
```

### 3. 著作权人身权 vs 财产权结构

```
著作权
├─ 人身权（恒久性、专属性，除发表权外无期限限制）
│   ├─ 署名权
│   ├─ 修改权
│   └─ 保护作品完整权
│
└─ 财产权（一般为50年）—— 共12项
    ├─ 复制权      ← 馆藏数字化实质
    ├─ 发行权      ← 首次销售原则相关
    ├─ 出租权
    ├─ 展览权
    ├─ 表演权
    ├─ 放映权
    ├─ 广播权
    ├─ 信息网络传播权  ← 数字图书馆核心争议
    ├─ 摄制权
    ├─ 改编权
    ├─ 翻译权
    └─ 汇编权      ← 数据库建设相关
```

**记忆口诀**：复制发行出租展览，表演放映广播网传，摄制改编翻译汇编，十二财产五十年。

### 4. CC协议核心元素与组合

```
CC核心元素（4个）：
┌─────────────────────────────────────────┐
│  by = 署名（Attribution）              │
│  nc = 非商业使用（Non-Commercial）      │
│  nd = 禁止演绎（No Derivative Works）   │
│  sa = 相同方式共享（Share-Alike）       │
└─────────────────────────────────────────┘

CC组合（6种）：
1. 署名（BY）
2. 署名-禁止演绎（BY-ND）
3. 署名-非商业使用（BY-NC）
4. 署名-非商业使用-禁止演绎（BY-NC-ND）
5. 署名-非商业性使用-相同方式共享（BY-NC-SA）
6. 署名-相同方式共享（BY-SA）
```

### 5. 合理使用障碍（3条）

```
障碍一：合理使用/法定许可范围有限
        └─ 《信息网络传播权保护条例》第七条相当苛刻

障碍二：作品权利保护期非常漫长
        └─ 1998年"米老鼠保护法案"延长20年

障碍三：一对一签约成本高、操作性差
        └─ 数字资源海量，逐一授权不现实
```

### 7. Copyright vs Copyleft vs CC 关系图

```
Copyright（著作权）
  │
  ├─ 默认：所有权利由作者保留（垄断性）
  │
  ├─ Copyleft（著佐权）
  │   └─ 提前声明，保留和让渡部分权利
  │      └─ 仍在Copyright框架中
  │      └─ 代表：GNU、GitHub
  │
  └─ CC（知识共享）
      └─ "部分权利保留"灵活模式
      └─ 4元素×6组合
      └─ CC0 = 完全放弃权利（公共领域）
```

## 【方法比较】

### 1. 金色OA vs 绿色OA

| 比较维度 | 金色OA（Gold OA） | 绿色OA（Green OA） |
|----------|-------------------|-------------------|
| **别称** | 开放出版（Open Access Publishing） | 开放存储（Open Access Archives） |
| **实现方式** | 通过发表在OA期刊上开放获取 | 存储到机构或学科知识库 |
| **两种形式** | 开放期刊（整刊开放）/开放论文（部分开放） | 机构知识库 / 预印本数据库 |
| **费用** | 可能有论文处理费APC | 一般免费自存储 |
| **版权归属** | 版权更多出现归属于作者 | 依据出版商自归档政策 |
| **开放时滞** | 立即开放 | 可能有 embargo（时滞期） |
| **政策依据** | SHERPA/RoMEO（确定存档版本和时滞） | SHERPA/RoMEO数据库 |
| **代表平台** | BMC、PLoS、DOAJ | arXiv、ChinaXiv、中国科技论文在线 |
| **中国平台** | GoOA、Socolar | OpenDOAR（资源库目录） |
| **核心区别** | 在OA期刊上发表 | 在知识库中存储已发表/未发表成果 |

<details>
<summary>点击查看答案与解析</summary>

**答案：**

**开放获取（Open Access, OA）**是指科研知识成果在互联网公开发布，用户可免费阅读、下载、复制、保存、传播、使用。同时著作权人保留著作权，通过开放共享许可协议（例如CC许可协议）授权所有用户开放使用。

| 维度 | 金色OA | 绿色OA |
|------|--------|--------|

| 开放时滞 | 立即开放 | 可能有embargo时滞期 |
| 代表平台 | BMC、PLoS、DOAJ | arXiv、ChinaXiv |

**解析：** 本题是高频名词解释题。答题时需先准确定义OA（强调"免费使用+保留著作权"两个关键点），再从实现方式、形式、费用、时滞等维度比较金色与绿色OA。注意金色OA是"发表在OA期刊上"，绿色OA是"存储到知识库"。

### 题目5（论述题）

**论述开放获取（OA）体系的演进历程及其对图书馆信息资源建设的意义。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

**一、OA体系演进历程**

1. **1994年**：美国认知科学教授斯蒂万·哈纳德首次提出"资源免费共享"（Open Access, OA）的理念。
2. **2002年**：《布达佩斯开放获取计划》发布，推动了科技文献的开放获取。
3. **2003年**：德国马普学会通过《关于自然科学与人文科学知识开放获取的柏林宣言》，将开放获取的对象扩展到科研论文、数据、参考资料、照片图表、学术类多媒体资源等，提出开放获取的两个条件。
4. **2016年**：德国马普学会等机构发起OA2020倡议，旨在将现有学术期刊从订购模式转换为开放出版模式。
5. **2018年**：11个欧洲国家的13个资助机构联合发布Plan S（cOAlition S），提出十项原则，要求2020年1月1日后公共资金资助成果必须开放获取。

**二、OA对图书馆信息资源建设的意义**

1. **丰富馆藏来源**：OA资源为图书馆提供了大量免费可获取的学术资源，拓展了馆藏来源渠道。
2. **缓解预算压力**：在期刊危机和预算紧缩背景下，OA资源可作为商业数据库的补充，降低采购成本。
3. **促进资源共享**：OA理念与图书馆共建共享理念高度契合，推动图书馆从"拥有"向"存取"转型。
4. **支持机构知识库建设**：绿色OA推动图书馆建设机构知识库，保存和传播本机构科研成果。
5. **应对合理使用障碍**：OA通过CC等开放许可协议，突破传统著作权限制，为数字资源建设提供法律保障。

**解析：** 本题是综合性论述题。需清晰呈现OA时间轴的五个关键节点，并从资源丰富、预算缓解、共享促进、知识库建设、法律保障等多角度分析意义。答题时注意逻辑递进，体现OA从理念提出到制度化的演进脉络。

### 题目6（案例分析题）

**阅读以下案例材料，回答问题。**

**案例：上海音乐出版社案**

原告上海音乐出版社有限公司、上海文艺音像电子出版社有限公司是录音制品《巴斯蒂安成人钢琴教程第一册》的制作者，对案涉制品享有信息网络传播权。被告广州联图电子科技有限公司未经许可通过其运营的网站"云图网"向公众提供案涉制品，且存在利用案涉制品进行商业盈利的情形。被告辩称其受多家图书馆委托构建案涉平台，为相关图书馆用户提供数字作品资源服务，应属合理使用。

广州互联网法院经审理认为，合理使用须限定于两方面条件：其一，服务对象为实名注册并获取电子借阅证等凭证的读者；其二，仅提供作品的浏览和阅读，不包含作品的下载。联图公司提供下载服务且未限定特定用户群体，超出合理使用范畴，构成侵权。

**问题：**
1. 本案的争议焦点是什么？
2. 法院认定的图书馆合理使用边界包含哪些条件？
3. 本案映射的信息资源建设知识点是什么？

<details>
<summary>点击查看答案与解析</summary>

**答案：**

**1. 争议焦点：** 图书馆委托技术服务商（联图公司）提供数字作品下载服务，是否属于合理使用范畴。

**2. 合理使用边界条件：**
- **条件一（对象限定）**：服务对象为实名注册并获取电子借阅证等凭证的读者，不能面向全体公众开放。
- **条件二（方式限定）**：仅提供作品的浏览和阅读，不包含作品的下载。下载本质上属于复制，一旦下载则图书馆无法控制传播范围。
- **补充**：搭建数字平台的相关技术提供者可以作为数字作品合理使用的适格主体，但不能超出图书馆合理使用的边界。

**3. 知识点映射：** 映射到3.3节"合理使用范围与限制"。本案明确了数字时代图书馆合理使用的边界——对象限定（实名注册读者）+方式限定（仅浏览不可下载），为图书馆数字化建设提供了司法指引。

**解析：** 本案（案号：(2020)粤0192民初36894号，广州互联网法院）是数字时代图书馆合理使用边界的标杆案例。答题时需注意区分合理使用的两个限定条件（对象+方式），并理解"技术提供者可作为适格主体但不可越界"这一延伸要点。

</details>

### 题目7（案例分析题）

**阅读以下案例材料，回答问题。**

**案例：AI文生图案**

原告李某某使用开源软件Stable Diffusion，通过输入正向、反向提示词，设置迭代步数、图片高度、提示词引导系数以及随机数种子等操作生成涉案图片后发布在小红书平台。被告刘某某在百家号发布文章，文章配图使用了涉案图片，且截去了原告在小红书平台的署名水印。原告认为被告侵犯了其署名权及信息网络传播权。

北京互联网法院认定：涉案图片符合作品的定义，属于美术作品，受著作权法保护；人工智能模型本身无法成为著作权法上的作者，原告基于智力投入直接产生并体现个性化表达，是涉案图片作者，享有著作权；被告去除署名水印并使用图片，侵害了署名权及信息网络传播权。

**问题：**
1. 法院认定AI生成内容构成作品的四个要件是什么？
2. 法院如何认定AI生成内容的著作权归属？
3. 本案映射的信息资源建设知识点是什么？

<details>
<summary>点击查看答案与解析</summary>

**答案：**

**1. AI生成内容构成作品的四个要件：**
- 是否属于文学、艺术和科学领域内（涉案图片属于艺术领域）
- 是否具有独创性（体现原告个性化表达，与在先作品存在可识别的差异性）
- 是否具有一定的表现形式（以线条、色彩构成的平面造型艺术作品）
- 是否属于智力成果（原告设计提示词、设置参数、选定图片体现了智力投入）

**2. 著作权归属认定：**
- 著作权法规定作者限于自然人、法人或非法人组织，**人工智能模型本身无法成为著作权法上的作者**。

### 题目8（案例分析题）

**阅读以下案例材料，回答问题。**

**案例：故宫博物院案**

原告北京胡天博胜数码信息技术有限公司主张其于2005年9月创作完成了明景德镇瓷器烧制的三维动画片产品，并交给《中国国家地理》杂志社的马泳宇。后原告发现被告故宫博物院陶瓷馆长时间播放该动画片，认为侵犯了其著作权。

被告故宫博物院辩称：其于2005年5月与《中国国家地理》杂志社签订《故宫古瓷研究中心动画制作合同》，约定该杂志社按照故宫博物院要求制作三维动画片，**动画片的著作权归故宫博物院所有**。第三人《中国国家地理》杂志社主张其与原告系租用设备和聘用人员的合作关系。

法院审理认为：故宫博物院与《中国国家地理》杂志社确曾签订涉案合同，双方约定故宫博物院委托该杂志社制作涉案动画片，合同约定"执行本合同产生的最终成果的所有权利"归属故宫博物院。该杂志社提交证据证明其向原告及其员工支付了设备租赁费用及劳动报酬。原告虽参与了制作过程，但未能充分举证证明涉案动画片系其自行创作完成。故驳回原告诉讼请求。

**问题：**
1. 本案的争议焦点是什么？
2. 法院如何认定涉案动画片的著作权归属？
3. 本案映射的信息资源建设知识点是什么？

<details>
<summary>点击查看答案与解析</summary>

**答案：**

**1. 争议焦点：** 在委托创作合同下，涉案三维动画片的著作权归属问题——是归属于实际参与制作的原告（胡天博胜公司），还是归属于委托方（故宫博物院）。

**2. 著作权归属认定：**
- 故宫博物院与《中国国家地理》杂志社签订了《故宫古瓷研究中心动画制作合同》，**合同明确约定"执行本合同产生的最终成果的所有权利"归属故宫博物院**。
- 第三人《中国国家地理》杂志社依据合同接受委托，在前期调研基础上组织原告人员及案外人共同创作完成，并租用原告设备，支付了设备租赁费和劳动报酬。
- 原告虽参与了制作过程，但**未能充分举证证明涉案动画片系其自行创作完成的作品**。
- 综上，著作权归属故宫博物院，被告使用涉案动画片未侵犯原告权利。

**3. 知识点映射：** 映射到3.3节"委托创作著作权权属"。本案明确了委托创作合同下著作权归属的认定规则——以合同约定为准，实际参与制作并不当然享有著作权。注意：本案是"委托创作著作权权属"案例，非"馆藏资源公共性冲突"案例（后者为故宫诉中国商业出版社案，课件正文已提及但无独立PDF）。

**解析：** 本案（案号：(2006)二中民初字第09960号，北京市第二中级人民法院，审结日期2006.11.07）是委托创作著作权权属的典型案例。核心要点："合同约定优先"+"参与制作≠享有著作权"。答题时注意区分本案（委托创作权属）与课件正文中提到的故宫诉中国商业出版社案（馆藏资源公共性冲突）。

</details>

### 题目9（案例分析题）【课件已引用案例】

**阅读以下案例材料，回答问题。**

**案例：广东省立中山图书馆案**

上诉人中国科技出版传媒股份有限公司（中科出版公司）是《内隐知识》一书的专有出版权和信息网络传播权被许可方。中科出版公司发现：通过北京邮电大学校园网进入读秀公司经营的"读秀网"，搜索《内隐知识》后点击"图书馆文献传递"，可跳转至中山图书馆主办的"全国参考资讯与文献传递网"，通过邮件获取该书全部内容（图书原版式扫描格式），包括在线和离线阅读、在线打印及下载保存。

一审法院认定读秀公司和中山图书馆构成信息网络传播权侵权，判决连带赔偿6000元。二审法院（北京知识产权法院）认为：读秀公司和中山图书馆未经许可通过网络提供涉案作品，构成信息网络传播权侵权；复制行为被信息网络传播行为吸收，不单独构成复制权侵权；网络传播不转移有形物质载体所有权，不构成发行权侵权；常规版式设计不享有版式设计权。但二审认定侵权行为非持续性（2014年8月15日邮件发送后即停止），中科出版公司2017年9月19日起诉已超过2年诉讼时效，故撤销一审判决，驳回全部诉讼请求。

**问题：**
1. 本案涉及哪些著作权权项的争议？法院如何逐一认定？
2. 二审法院为何改判驳回全部诉讼请求？
3. 本案映射的信息资源建设知识点是什么？

<details>
<summary>点击查看答案与解析</summary>

**答案：**

**1. 著作权权项争议及法院认定：**

| 权项 | 法院认定 | 理由 |
|------|----------|------|
| **信息网络传播权** | 构成侵权 | 读秀公司和中山图书馆未经许可通过分工合作提供涉案作品，使公众可在选定时间地点获得 |
| **复制权** | 不单独构成 | 复制行为被信息网络传播行为吸收，在信息网络传播权控制范围之外未产生新的复制行为 |
| **发行权** | 不构成 | 网络传播不能导致作品有形物质载体所有权的转移 |
| **版式设计权** | 不享有 | 涉案作品在用字、行距、标点、编排格式等方面属常规出版物版式设计，无法体现设计安排 |

**2. 二审改判理由：**
- 二审认定涉案侵权行为**非持续性侵权行为**：2014年8月15日"全国参考咨询与文献传递网"通过发送电子邮件将涉案作品发送后即行停止，邮件内容有效期为20天，20天内只允许打开20次。
- 中科出版公司公证保全时间为2014年8月15日，向法院提交起诉状时间为2017年9月19日，**已超过2年诉讼时效期间**，且无中断、延长等事由。
- 故二审撤销一审判决，驳回中科出版公司全部诉讼请求。

**3. 知识点映射：** 映射到3.3节"馆藏资源提供中的信息网络传播权问题"。本案（课件已引用案例）涉及数字图书馆扫描图书提供文献传递是否侵权的问题，明确了信息网络传播权、复制权、发行权、版式设计权在网络环境下的不同认定标准，以及诉讼时效在著作权侵权中的适用。案号：(2019)京73民终3341号，北京知识产权法院二审，审结日期2020.03.25。

**解析：** 本案是课件正文已配案例，用以说明"馆藏资源提供中的信息网络传播权问题突出"。答题时需注意四个权项的逐一认定（信息网络传播权侵权，复制/发行/版式设计不侵权），以及二审以诉讼时效为由改判的特殊性。

</details>

### 题目10（论述题）

**论述图书馆面临的预算压力与期刊危机，并分析图书馆的应对策略。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

1. **程焕文"八问"**：质疑数据库定价依据不明、强行捆绑纸质期刊、为80%未被使用资源买单、涨价幅度无规矩、数字图书价格高于纸本、购买使用权后不能拥有永久使用权、高影响因子期刊单独建库暴利销售等。

3. **缩减其他资源购买**：在预算有限时调整采购结构。
4. **停购数据库**：对不合理涨价的数据库采取停购策略。
5. **新道路——开放存取与机构知识库**：利用OA资源补充馆藏，建设机构知识库保存和传播本机构成果。

### 题目12（简答题）

**简述CC协议（知识共享许可协议）的核心元素、组合方式及其对图书馆的意义。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

**CC协议核心元素（4个）：**
- **by（署名/Attribution）**：使用者须以适当方式表明原作者
- **nc（非商业使用/Non-Commercial）**：不得用于商业目的
- **nd（禁止演绎/No Derivative Works）**：不得改动、转换作品
- **sa（相同方式共享/Share-Alike）**：演绎作品须以相同协议发布

**CC组合（6种）：**
1. 署名（BY）
2. 署名-禁止演绎（BY-ND）
3. 署名-非商业使用（BY-NC）
4. 署名-非商业使用-禁止演绎（BY-NC-ND）
5. 署名-非商业性使用-相同方式共享（BY-NC-SA）
6. 署名-相同方式共享（BY-SA）

**CC协议对图书馆的意义：**
1. **提供授权模式**：CC在传统著作权"保留所有权利"与"不保留任何权利"之间，使创作者可"保留部分权利"，为图书馆提供灵活的授权选择。
2. **丰富数字资源**：图书馆可将"部分让渡权利"的作品系统化加工整合，保障作者基本著作权的同时，增加为社会公众服务的整体资源，合理合法保存人类文化遗产。

**解析：** 本题考查CC协议体系。需准确记忆4核心元素（中英文）和6组合。对图书馆的意义需从"授权模式"和"资源丰富"两个维度作答。注意CC0协议是完全放弃权利（公共领域），与6种组合不同。

</details>
