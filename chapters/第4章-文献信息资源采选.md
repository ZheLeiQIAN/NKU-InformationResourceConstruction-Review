## 【核心概念】

### ★ 必考概念

| 概念 | 定义 | 关键要点 |
|------|------|----------|
| **文献采访** | 图书馆最重要的基础工作之一，是信息资源建设的首要环节，包括文献的选择与采集两个环节 | 采+访两环节，"访"在"采"前 |
| **文献选择（Selection）** | 图书馆遵循一定方针、原则、标准，对众多文献进行鉴别、判断，从中挑选出适合馆藏建设目标及用户需要的文献的过程 | 强调"鉴别判断"和"挑选" |
| **文献采集（Acquisition）** | 主动寻找文献源，利用各种途径，采用多种方式方法，按一定程序和规则采集文献，以保证收集到已选定的出版物 | 强调"主动寻找"和"保证获取" |
| **五级等级结构** | 完整级→研究级→大学级→基础级→最低级，由高到低的藏书建设等级体系 | 五级递降，完整级最高、最低级在范围之外 |
| **完整级** | 完整齐全搜集某专题领域所有知识记录为目标的藏书方案 | 最高级别，国家图书馆核心领域适用 |
| **研究级** | 以满足独立研究需要为目标的藏书方案 | 特藏应达到研究级 |
| **大学级** | 以满足大学生和个人自学大学课程的需要为目标的藏书方案 | 高校图书馆基础配置 |
| **基础级** | 以介绍人们认识不同专业领域为目标的藏书方案 | 普及性，县级馆适用 |
| **最低级** | 在藏书范围之外的藏书方案 | 非馆藏重点，最低层次 |
| **SCI（Science Citation Index）** | 美国科学信息研究所（ISI）1961年创办的引文数据库，覆盖生命科学、临床医学、物理化学等，反映自然科学研究学术水平 | 1961年创办，自然科学 |
| **SSCI（Social Sciences Citation Index）** | ISI建立的综合性社科文献数据库，涉及经济、法律、管理、心理学等，收录50个语种1700多种期刊 | 社科领域，50语种1700种 |
| **CSCD（中国科学引文数据库）** | 中国科学院国家科学图书馆制作，收入中英文科技核心期刊近千种，核心库670种、扩展库378种，被誉为"中国的SCI" | "中国的SCI"，670+378种 |
| **CSSCI（中文社会科学引文索引）** | 南京大学承担的教育部重大研究项目，从全国3500种中文人文社科期刊中精选来源期刊，定量与定性相结合 | 南大，3500种中精选 |
| **北大核心（中文核心期刊要目总览）** | 北京大学图书馆与北京高校图书馆期刊工作研究会共同编撰，评价指标由7个指标组成，统计源52种数据库或文摘期刊 | 7指标，52种统计源 |
| **读者决策采购（PDA）** | 将读者的阅读需求量化成一定指标，作为图书馆文献采购和馆藏建设的决策依据，也称DDA、POD | 需求驱动，电子书时代触发购买 |
| **纲目订书** | 图书馆将藏书补充计划按入藏文献范围、内容、出版社、价格等列成纲目提交给书商，书商按"纲目"直接寄书，退书率不得高于10% | 退书率≤10%，签一次合同连续供书 |

### ☆ 重要概念

| 概念 | 定义 | 关键要点 |
|------|------|----------|
| **SCIE（Science Citation Index Expanded）** | 科学引文索引扩大版，SCI的扩展版本 | 比SCI收录范围更广 |
| **EI（Engineering Index）** | 美国工程索引，工程技术领域权威检索工具 | 工程技术领域 |
| **CA（Chemical Abstracts）** | 美国化学文摘 | 化学领域 |
| **SA（Science Abstracts）** | 英国科学文摘 | 英国，综合科学 |
| **特藏** | 按某一专题（地域、人物、时间、事件、机构、学科等）、某一特殊原因而专门收藏的文献资料集合 | 具有独特性，研究级 |
| **交存（缴送/呈缴）制度** | 出版机构向指定保存机构送交其出版物样本的制度，确保国家和地方文化遗产完整、系统保存 | 清朝起源，2017公共图书馆法第26条 |
| **图书复本** | 泛指两本以上，内容和形式完全相同的重复书；第二册以后称为复本，或指收藏同一种出版物的册数 | 品种=种数，复本=册数 |
| **品种与复本** | 品种表现不同内容图书的数量（种数）；复本表现相同内容图书的重复数量（册数/本数/件数） | 经费有限时两者矛盾 |
| **种多册少** | 图书品种多、每种复本少，适用于科研用书和专业性科研图书馆 | 多样性优先 |
| **种少册多** | 图书品种少、每种复本多，适用于指定主要教学参考用书 | 满足集中需求 |
| **集中式交换** | 将可供交换文献送交交换中心，由中心集中交换，最基本最常用方式 | 非营利，仅收少许手续费 |
| **长期订购** | 保证多卷和连续出版物完整入藏的手段，一次手续长期有效，可享10%-30%优惠 | 连续出版物，优惠10%-30% |

## 【重点公式/算法】

> 本章以制度框架和分类体系为主，以下为需记忆的关键结构、数字和分类规则。

### 1. 文献采访两环节结构

```
文献采访（Acquisitions）
    │
    ├─ 文献选择（Selection）  ← 先：鉴别、判断、挑选
    │   "遵循方针原则标准，从众多文献中挑选适合馆藏目标和用户需要的文献"
    │
    └─ 文献采集（Acquisition） ← 后：寻找、采集、保证获取
        "主动寻找文献源，利用各种途径方法，采集已选定的出版物"
```

**记忆口诀**：先选后采，访在采前；选是判断，采是行动。

### 2. 五级等级结构（由高到低）

```
完整级  ★★★  ── 完整齐全搜集某专题领域所有知识记录
  │           （国家图书馆核心领域）
  ▼
研究级  ★★   ── 满足独立研究需要
  │           （特藏、重点学科应达到此级）
  ▼
大学级  ★    ── 满足大学生和个人自学大学课程需要
  │           （高校图书馆基础配置）
  ▼
基础级       ── 介绍人们认识不同专业领域
  │           （普及性，县级馆适用）
  ▼
最低级       ── 在藏书范围之外
              （非馆藏重点）
```

**记忆口诀**：完研大基最，五级从高降；国家求完整，特藏达研究。

### 3. 国家图书馆采选方针与四级别

```
采选方针："中文求全"、"国内出版物求全"与"外文求精"

采选四级别：
├─ 全面采选   ← 中文各学科各类型文献全面入藏
├─ 着重采选   ← 古籍善本、名人手稿、珍稀文献
├─ 适当采选   ← 外文文献适时调整范围和重点
└─ 不宜采选   ← 范围之外文献
```

### 4. 图书采访一般标准（6要素）

```
（1）内容范围与读者对象 ── 信息容量、多层次需求、时效性
（2）内容质量          ── 篇幅、创新程度、准确性、专业针对性
（3）责任者            ── 著者编者的知名度及社会评价
（4）出版者与出版记录   ── 出版社声誉、出版年代/版次/定价
（5）价格              ── 昂贵书需提交采访委员会决定
（6）印刷质量           ── 现购时翻阅检查
```

**记忆口诀**：内容质量责任者，出版价格印刷质，六要素看图书。

### 5. 北大核心期刊7指标体系

```
北京大学图书馆《中文核心期刊要目总览》评价指标：

（1）被索量    ── 被检索系统收录频次
（2）被摘量    ── 被文摘刊物摘录频次
（3）它引量    ── 被其他论文引用次数
（4）影响因子  ── 近两年论文平均被引次数
（5）被摘率    ── 被摘量与发文量之比
（6）获奖      ── 获国家级/省部级奖项
（7）被重要检索工具收录 ── 被SCI/EI/CSSCI等收录

统计源：52种数据库或文摘期刊
```

**记忆口诀**：索摘引影摘奖收，七指标定核心；五十二源作统计。

### 6. 读者决策采购（PDA）4种类型

```
按读者决策方式（参与途径）划分：

（1）通过馆际互借      ── 对互借请求购买而非借阅（普渡大学，150美元/3年内）
（2）通过馆藏/联合目录  ── 书目导入目录，读者检索后荐购（伊利诺伊，300美元/1年内）
（3）通过电子书点击浏览  ── 浏览触发自动购买（丹佛大学，5分钟/10页→短期租借→3次购买）
（4）通过网上/线下书店   ── 读者书店发现→图书馆购买（水牛城大学，IP认证）
```

### 7. 非购买采集方式分类

```
非购入方式（不用货币购买，免费或少量经费获取）
    │
    ├─ 交存（缴送/呈缴） ── 依据交存制度获取出版物
    ├─ 征集              ── 对非正式出版单位内部书刊主动征集
    ├─ 调拨              ── 主管部门组织，有计划调拨馆藏
    ├─ 捐赠（赠送）       ── 机构/个人主动赠送
    ├─ 交换              ── 双边式 / 多边式 / 集中式
    └─ 索取              ── 通讯联系或派人上门取回
```

### 8. 文献交换三种形式

```
（1）双边式交换
    A ◄────直接寄送────► B
    范围广、周期长、不确定性高

（2）多边式交换
    各馆 ──可供文献/需求──► 交换书目中心 ──分发供求书目──► 各馆
    麻烦、交换对象不确定、跨国费用高

（3）集中式交换 ★（最基本最常用）
    各馆 ──可供文献──► 文献交换中心 ──集中交换──► 各馆
    简单易行、周期短、针对性强、非营利、仅收少许手续费
```

### 9. 文献采集四阶段流程

```
选书阶段 ──────► 订购阶段 ──────► 验收阶段 ──────► 补缺阶段
   │                │                │                │
   ▼                ▼                ▼                ▼
搜集书目信息    采购行动实施    财产验收比对      催缺未到图书
审查圈选        办理审批手续    实体核实          催缺仍不到馆
查重            填报预订单      财产分配/登记     撤销原有订单
                寄发订购单      经费统计支出      其他渠道重订
```

### 10. 选择馆配商5种方式

```
（1）公开招标      ← 中文图书
（2）邀请招标
（3）竞争性谈判    ← 外文图书、外文期刊
（4）询价采购
（5）单一来源采购
（6）采购备案      ← 数据库
```

**实务映射**：中文图书→公开招标；外文图书期刊→竞争性谈判；数据库→采购备案。

### 11. 确定复本量8因素

```
（1）现实读者数      ── 成正比（同时需求同一文献的读者数）
（2）图书借阅制度    ── 借阅方式/期限/数量影响大
（3）图书的有效期    ── 有效期长→复本少
（4）图书的类型与价格 ── 类型不同需求不同，价格受经费限制
（5）流通部门数与保存本数 ── 成正比
（6）图书馆复制能力  ── 复制能力强→复本量可低
（7）数字化资源数量  ── 数字化→减少纸质复本
（8）其他因素        ── 文种通用性、使用范围、水平深浅
```

**记忆口诀**：读者借阅有效期，类型价格流通部，复制数字其他因，八条定复本。

## 【方法比较】

### 1. 各类图书馆采选定位对比 ★★★

| 维度 | 国家图书馆 | 公共图书馆 | 高校图书馆 | 科学专业图书馆 |
|------|-----------|-----------|-----------|--------------|
| **核心职能** | 获取和保存本国出版物，编制国家书目 | 提供普遍均等服务，满足大众阅读 | 为教学和科学研究服务 | 传递科学情报，服务科研 |
| **采选方针** | "中文求全""外文求精" | 综合性、通用性、普及性 | 教育性、学术性、专业性、先进性、系统性完整性 | 目的性、针对性、研究性、先进性时代性 |
| **采选任务** | 全面系统收集保存国家文献，成为本国书刊查询借阅最终基地 | 省：地区综合性科学藏书；市：实用价值文献精选；县：普及性实用文献 | 重点系统入藏专业及教学科研用书，保证完整性和连续性 | 购置科研人员所需科技文献，建立高质量专业特色馆藏 |

### 题目1（名词解释）

**请解释"文献采访"的概念，并说明其包含的两个环节。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

**文献采访**是图书馆最重要的基础工作之一，是信息资源建设的首要环节。文献采访主要包括**文献的选择**与**文献的采集**两个重要环节。

**文献选择（Selection）**：图书馆遵循一定的方针、原则、标准，对众多的文献进行鉴别、判断，从中挑选出适合馆藏文献资源建设目标及用户需要的文献的过程。

**文献采集（Acquisition）**：主动地寻找文献源，利用各种途径，采用多种方式方法，并按照一定的程序和规则采集文献，以保证收集到那些已经选定的出版物。

**解析**：本题是高频名词解释题。答题时需先给出文献采访的整体定义，再分别阐述"选择"和"采集"两个环节的定义。注意"选择"强调"鉴别判断和挑选"，"采集"强调"主动寻找和保证获取"。两者是先后关系，先选后采。

**简述各类图书馆（国家图书馆、公共图书馆、高校图书馆、科学专业图书馆）的文献采选定位。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

**一、国家图书馆**

### 题目4（简答题）

**简述国内外主要期刊评价体系。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

期刊评价体系是从文献之间的引证关系着手，揭示科学文献之间内在联系的评价系统，通过文献计量法分析学科研究动态、核心作者群，评价学术期刊的学术水平和地位。

**一、国际期刊评价体系**

1. **SCI（科学引文索引）**：美国科学信息研究所（ISI）1961年创办，覆盖生命科学、临床医学、物理化学、农业、生物、工程技术等，尤其能反映自然科学研究的学术水平
2. **SCIE（科学引文索引扩大版）**：SCI的扩展版本，收录范围更广
3. **SSCI（社会科学引文索引）**：ISI建立的综合性社科文献数据库，涉及经济、法律、管理、心理学等，收录50个语种1700多种期刊，累计约350万条记录
4. **EI（工程索引）**：美国工程索引，工程技术领域权威检索工具
5. 其他：CA（美国化学文摘）、SA（英国科学文摘）、日本科学技术文献速报、俄国文摘杂志

**二、国内期刊评价体系**

1. **CSCD（中国科学引文数据库）**：中国科学院国家科学图书馆制作，收入中英文科技核心期刊近千种，核心库来源期刊670种、扩展库378种，被誉为"中国的SCI"
2. **CSSCI（中文社会科学引文索引）**：南京大学承担的教育部重大研究项目，从全国3500种中文人文社科期刊中精选来源期刊，定量与定性相结合
3. **北大核心（中文核心期刊要目总览）**：北京大学图书馆与北京高校图书馆期刊工作研究会共同编撰，评价指标由7个指标组成（被索量、被摘量、它引量、影响因子、被摘率、获奖或被重要检索工具收录），统计源52种数据库或文摘期刊
4. **自然科学学术期刊评价指标体系**：中国科学院自然科学期刊编辑研究会承担，2002年1月通过专家评审

### 题目7（简答题）

**简述文献信息资源采选的流程及各阶段任务。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

文献信息资源采选流程包括四个阶段：

**一、选书阶段**
- 搜集各种书目信息
- 将征订目录送交有关人员进行审查圈选
- 将初步审查圈选的征订目录进行核对"查重"

**二、订购阶段**
- 是初选和查重之后的采购行动实施过程
- 通过多种方式采选图书
- 办理相关审批手续
- 填报预订单、寄发订购单，完成订购工序

**三、验收阶段**
- 针对图书订购记录中的信息进行，是图书实体特征和订购信息的比对
- 实际上是财产验收
- 涉及图书实体核实、财产分配、财产登记、财产经费统计和支出
- 验收方式：按发票验收、提样本核对预订、财务报销

**四、补缺阶段**
- 对未到图书进行催缺
- 催缺后仍无法到馆，可视具体情况撤销原有订单
- 及时通过其他方式和渠道对图书再行订购

**解析**：本题是简答题和排序题考点。答题时需按顺序列出四阶段，并说明各阶段的核心任务。注意"查重"在选书阶段，"财产验收"在验收阶段。

</details>

### 题目8（论述题）

**论述确定藏书复本量的主要依据，并分析不同类型图书馆应采取的复本策略。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

**一、确定藏书复本量的8个依据**

1. **现实读者数**：指在同一时间需求同一种文献的本馆读者数。复本量与现实读者数量成正比。
2. **图书借阅制度**：图书馆规定的图书借阅方式、期限及数量等制度对复本量的影响较大。
3. **图书的有效期**：有效使用期长的图书所需要的复本比有效使用期短的图书要少。
4. **图书的类型与价格**：读者对不同类型图书使用目的不同，复本需求不同；受经费限制需考虑价格因素。
5. **流通部门数与保存本数**：图书馆流通分支部门数量、保存本数量与复本数量成正比。
6. **图书馆复制能力**：如能以较低价格复制读者所需图书，即使复本量低也能满足需求。
7. **数字化资源的数量**：文献数字化和网络资源的提供利用，能减少纸质文献的复本量。
8. **其他因素**：如出版物的文种通用与否、使用范围宽窄、水平深浅难易程度等。

**二、不同类型图书馆的复本策略**

1. **专业性科研图书馆**：用户研究课题用书需要广泛分散，藏书应做到"种多册少"。
2. **省级公共图书馆**：要满足研究型读者和大众型读者两类不同用书需要——科研用书"种多册少"，大众读物"种多册多"。
3. **高等学校图书馆**：读者需求复杂多样，应分别采取不同办法：
   - 科研用书："种多册少"
   - 指定主要教学参考用书："种少册多"
   - 教学参考用书及重要课外读物："种多册多"
   - 一般参考用书及一般课外读物："种少册少"

**三、品种与复本的关系处理**

在经费有限的条件下，增加图书品种和增加图书复本往往产生矛盾。处理这一问题的关键策略是：
- 品种表现不同内容图书的数量（种数），复本表现相同内容图书的重复数量（册数）
- 多数馆采取低复本策略以充分利用资金
- 注意利用图书品种对于复本的替代关系
- 各类型图书馆应从实际出发，因馆而异、因书而异、因需而异

**解析**：本题是综合性论述题。答题时需先列举8个依据，再分析不同类型图书馆的策略，最后落脚到品种与复本关系的处理原则。注意四种策略（种多册少/种少册多/种多册多/种少册少）的适用场景是核心得分点。

</details>

### 题目9（名词解释）

**请解释"读者决策采购"（PDA）的概念及其主要类型。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

**读者决策采购（Patron Driven Acquisitions, PDA）**，也称"按需购买"（Purchase-on-Demand, POD）、"需求驱动采购"（Demand-Driven Acquisitions, DDA），是指将读者的阅读需求量化成一定指标，作为图书馆文献采购和馆藏建设的决策依据。PDA源于利用馆际互借服务，促进、补充纸本馆藏建设；在电子书时代，它特指图书馆基于读者的实际浏览与阅读情况，以是否达到一定的标准或参数触发对某一文献购买指令的资源建设新模式。

**主要类型（按读者决策方式划分）：**

1. **通过馆际互借**：对读者通过馆际互借系统提出的请求，采用购买而不是互借的方式满足。（如普渡大学：近三年出版、150美元以内、一周内可供货）
2. **通过馆藏目录或联合目录**：将书商书目数据导入馆藏目录或联合目录，读者检索到未收藏图书时提出购买请求。（如伊利诺伊大学：过去一年内出版、300美元以下英语学术图书）
3. **通过电子图书的点击与浏览**：通过读者对电子图书的点击与浏览自动触发购买。（如丹佛大学：免费浏览5分钟或10页→短期租借→3次触发自动购买）
4. **通过网上/线下书店**：读者在网上或线下书店检索到图书后提出购买请求。（如水牛城大学：通过IP地址认证，符合预设标准即提示图书馆购买）

**解析**：本题是名词解释和简答题考点。需先给出PDA的定义及别名（PDA/DDA/POD），再列举4种类型。注意PDA的核心是"需求驱动"，从馆际互借起源，在电子书时代发展为"触发购买"模式。

</details>

### 题目10（判断/辨析题）

**判断以下说法是否正确，并说明理由：**

（1）文献交换中，多边式交换是最基本和最常用的方式。

（2）长期订购可以享受10%-30%的优惠，是保证多卷和连续出版物完整入藏的手段。

（3）国家图书馆采选外文文献时应遵循"外文求全"的原则。

<details>
<summary>点击查看答案与解析</summary>

**答案：**

（1）**错误**。集中式交换才是文献交换最基本和最常用的方式。多边式交换比较麻烦，交换对象不确定，且跨国交换费用较高。

（2）**正确**。长期订购只须办理一次手续，长期有效，而且还可享受10%-30%的优惠，是保证多卷和连续出版物完整入藏的一种手段。

（3）**错误**。国家图书馆的采选方针是"中文求全"、"国内出版物求全"与"外文求精"。外文文献应遵循"求精"而非"求全"原则，即根据本馆方针、任务，有重点、有选择地采选有关外文书刊资料。

**解析**：本题考查对采选方式和采选方针的理解。易错点在于：将集中式与多边式交换混淆；将"外文求精"误记为"外文求全"。国家图书馆的方针是"中文求全、外文求精"，二者不可混淆。

</details>
