## 【核心概念】

### ★ 必考概念

| 概念 | 定义 | 关键要点 |
|------|------|----------|
| **信息资源（霍顿）** | 两层含义：单数指信息内容本身；复数指支持工具（供给、设备、环境、人员、资金等） | 记住"单数内容、复数工具" |
| **信息资源（乌家培）** | 狭义：信息内容本身；广义：还包括信息设备、人员、系统、网络 | 狭义vs广义的区分是常考点 |
| **ISBN** | 国际标准书号，13位数字，前三位代表图书，中间9位分三组，最后一位校验码 | 13位结构；与ISSN区分 |
| **ISSN** | 国际标准刊号，8位数字编码，唯一标识印刷型期刊 | 8位；与ISBN区分 |
| **劫持性期刊（Hijacked Journal）** | 通过伪造期刊名称、模仿官网等方式冒充正规学术期刊的犯罪行为 | 概念较新，注意名词解释 |
| **专利权三属性** | 独占性、时间性、地域性 | 常考多选题 |
| **专利三要求** | 新颖性、创造性、实用性 | 常考多选题 |
| **信息资源建设根本目的** | 信息资源的利用（收藏和保存只是前提） | 理念题高频考点 |
| **"拥有"与"存取"** | 拥有=文献型所有权；存取=数字化使用权；两者相互辅助、长期共存 | 关系辨析是论述题热点 |

### ☆ 重要概念

| 概念 | 定义 | 关键要点 |
|------|------|----------|
| **经济资源（Economic Resource）** | 一切直接或间接为人类所需并构成生产要素的、稀缺的、具有开发利用选择性的资源 | 四大资源：自然、人力、资本、信息 |
| **连续出版物** | 具有统一名称、固定版式、统一开本、连续编号，汇集多位著者多篇著述，定期/不定期编辑发行的出版物 | 与图书的本质区别 |
| **EISSN（Electronic International Standard Serial Number）** | 电子国际标准连续出版物编号 | 电子版的ISSN |
| **CN号（China Number）** | 国内统一刊号，格式CN XX-XXXX/YY | 中国期刊的身份标识 |
| **美国四大科技报告** | PB报告（Publication Board）、AD报告（ASTIA Documents）、NASA报告（National Aeronautics and Space Administration）、DOE报告（Department of Energy） | 记住缩写全称 |
| **专利类型与期限** | 发明专利20年、实用新型10年、外观设计10年 | 期限数字易考 |
| **专利许可方式** | 普通许可、排他许可、独占许可、强制许可、交叉许可 | 五种方式 |
| **数据→信息→知识→智慧** | 数据层级递进关系 | 理解层级跃迁逻辑 |
| **信息资源建设定义** | 人类对无序媒介信息进行选择、采集、组织和开发，形成可利用信息资源体系的全过程 | 四个动词是核心 |
| **"信息沼泽"** | 信息繁杂无序、价值差异明显的现象 | 形象化表述 |
| **数字鸿沟（Digital Divide）** | 信息成为资源后，竞争与共享并存导致的信息获取不平等 | 社会热点关联 |

## 【重点公式/算法】

> 本章以概念为主，公式较少，以下为需要记忆的关键数字和编码规则。

### 1. ISBN 编码结构（13位）

```
ISBN: 978-7-302-12345-6
      │   │  │   │     │
      │   │  │   │     └─ 校验码（1位）
      │   │  │   └─ 书序号（出版社内编号）
      │   │  └─ 出版社号
      │   └─ 组号（国家/地区/语言区）
      └─ 图书产品标识（978/979）
```

**记忆口诀**：图书978，三组中间夹，末位校验码。

### 2. ISSN 编码结构（8位）

```
ISSN: 1234-5678
      │      │
      │      └─ 校验码（1位，可能是X）
      └─ 前7位数字
```

**与ISBN核心区别**：ISSN是8位，无分组含义，用于连续出版物；ISBN是13位，用于图书。

### 3. 全国统一书号结构（1987年前使用）

```
全国统一书号 = 图书分类号 + 出版社代号 + 出版序号
```

### 4. CN号格式

```
CN XX-XXXX/YY
│  │   │   │
│  │   │   └─ 分类号
│  │   └─ 地区连续出版物的序号
│  └─ 地区号
└─ 中国国别代码
```

### 5. 专利保护期限

| 专利类型 | 保护期限 |
|----------|----------|
| 发明专利 | 20年 |
| 实用新型 | 10年 |
| 外观设计 | 10年 |

## 【方法比较】

### 1. 霍顿 vs 乌家培：信息资源定义比较

| 维度 | 霍顿（F.W.Horton） | 乌家培 |
|------|---------------------|--------|
| **区分方式** | 单数/复数语法区分 | 狭义/广义范围区分 |
| **单数/狭义** | 信息内容本身 | 信息内容本身 |
| **复数/广义** | 支持工具（供给、设备、环境、人员、资金） | 信息设备、人员、系统、网络 |
| **核心差异** | 强调"支持条件" | 强调"系统要素" |
| **考试提示** | 常考"两层意思"的表述 | 常考"狭义vs广义"的辨析 |

### 2. 图书 vs 连续出版物

| 比较维度 | 图书 | 连续出版物 |
|----------|------|------------|
| **页数标准** | ≥49页（ISO/UNESCO） | 无固定页数要求 |
| **编号系统** | ISBN（13位） | ISSN（8位） |
| **出版周期** | 一次性出版 | 定期/不定期连续出版 |
| **著者特点** | 通常单一或少数著者 | 汇集多位著者多篇著述 |
| **版本特点** | 有版次（第1版、第2版） | 有卷期号（Vol.1 No.1） |
| **示例** | 教材、专著 | 期刊、报纸、年鉴 |

### 3. 文献信息资源四大类型比较

| 类型 | 记录手段 | 载体 | 优点 | 缺点 |
|------|----------|------|------|------|
| **刻写型** | 刻划、手工书写 | 自然物质、纸张 | 原始性、真实性 | 易损、难以复制 |
| **印刷型** | 印刷技术 | 纸张 | 阅读方便、流传广泛 | 体积大、存储空间有限 |
| **缩微型** | 光学记录 | 感光材料 | 体积小（省98%空间）、保存期长（500年） | 需借助机器阅读 |
| **视听型** | 电磁记录 | 电磁材料 | 动态直观、多媒体 | 需专用设备 |

### 4. 信息资源建设三阶段演变

| 阶段 | 时间 | 核心概念 | 特征 |
|------|------|----------|------|
| **藏书建设** | 20世纪50年代 | 藏书 | 强调收集、保管、复选、剔除 |
| **文献资源建设** | 20世纪80年代 | 文献资源 | 强调系统规划、建立藏书体系 |
| **信息资源建设** | 20世纪90年代 | 信息资源 | 强调多种媒介、数字化、共建共享 |

**演变逻辑**：范围扩大（图书→文献→信息）→ 载体扩展（纸质→多种媒介）→ 理念升级（收藏→利用→共享）

### 5. 数字信息资源分类方式比较

| 分类维度 | 类别 | 说明 |
|----------|------|------|
| **存储方式** | 网络信息资源 / 单机信息资源 | 是否联网访问 |
| **表现形式** | 文本类 / 数据类 / 多媒体类 | 信息呈现形式 |
| **生成方式** | 数字化信息资源 / 原生型数字资源 | 是否由纸质转化而来 |

### 6. 数字信息资源主要类型细分

| 类型 | 英文全称 | 说明 | 示例 |
|------|----------|------|------|
| **数据库（Database）** | Database | 按一定结构组织、存储和管理的数据集合，可检索、可共享 | 中国知网（CNKI）、万方、Web of Science、PubMed |
| **电子期刊（E-journal）** | Electronic Journal | 以数字形式出版、发行的连续性出版物，具有ISSN | Elsevier ScienceDirect、SpringerLink、IEEE Xplore |
| **电子图书（E-book）** | Electronic Book | 以数字形式出版的图书，具有ISBN | 超星数字图书馆、Springer电子图书、掌阅 |
| **电子报纸（E-newspaper）** | Electronic Newspaper | 以数字形式出版、定期发行的报纸 | 人民日报数据库、ProQuest Historical Newspapers |
| **学位论文库** | Dissertation/Thesis Database | 收录硕士、博士学位论文的数据库 | 中国博士学位论文全文数据库（CDFD）、ProQuest Dissertations & Theses |
| **会议论文库** | Conference Proceedings Database | 收录学术会议论文的数据库 | 中国重要会议论文全文数据库（CPCD）、ACM Digital Library |
| **专利数据库** | Patent Database | 收录专利文献的数据库 | 中国专利全文数据库、Derwent Innovations Index |
| **标准数据库** | Standards Database | 收录各类标准文献的数据库 | 国家标准全文数据库、ISO/IEC标准库 |
| **多媒体资源** | Multimedia Resource | 包括音频、视频、图像等非文本资源 | 超星学术视频、Khan Academy、MOOC课程 |
| **网络信息资源** | Network Information Resource | 通过互联网免费或授权获取的信息资源 | 政府网站、开放获取期刊（OA）、机构知识库 |

**考试提示**：数字信息资源的分类是高频考点，注意区分"数据库"（结构化数据集合）与"电子期刊/电子图书"（具体文献类型）的层级关系。

### 题目1（名词解释）

**请解释"信息资源"的概念，并比较霍顿与乌家培两种定义的区别。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

**信息资源**是经过人类选择、组织和加工处理的有序化的各种媒介信息的集合。

**霍顿的定义**（两层含义）：
- 单数（information resource）：指信息内容本身
- 复数（information resources）：指支持工具，包括供给、设备、环境、人员、资金等

**乌家培的定义**（狭义与广义）：
- 狭义：信息内容本身
- 广义：还包括信息设备、信息人员、信息系统、信息网络等

**区别**：霍顿以语法单复数区分，强调"内容"与"支持条件"；乌家培以范围大小区分，强调"内容"与"系统要素"。

**解析**：本题是高频名词解释题。答题时需分别写出两种定义，并指出核心差异。注意霍顿的"单数/复数"和乌家培的"狭义/广义"是各自的标志性表述。

</details>

### 题目2（填空/选择）

**ISBN由____位数字组成，其中前三位代表____；ISSN由____位数字组成，用于标识____。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

ISBN由 **13** 位数字组成，其中前三位代表 **图书**；ISSN由 **8** 位数字组成，用于标识 **印刷型期刊**（或连续出版物）。

**解析**：ISBN和ISSN的位数是必考数字题。ISBN为13位（2007年后统一为13位），ISSN为8位。注意区分两者的适用对象：ISBN用于图书，ISSN用于连续出版物（期刊、报纸等）。

</details>

### 题目3（简答题）

**简述信息资源建设"拥有"与"存取"的关系。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

**"拥有"**指文献型信息资源的所有权，即图书馆实际收藏纸质文献。

**"存取"**指数字化信息资源的使用权，即通过网络获取电子资源的使用权限。

**两者关系**：
1. **相互辅助**：纸质资源与数字资源互为补充，满足不同场景需求
2. **长期共存**：在可预见的未来，两种模式将同时存在
3. **缺一不可**：仅有拥有无法满足海量信息需求，仅有存取无法保障核心资源稳定获取
4. **互动关系**：拥有为存取提供基础，存取拓展拥有的服务边界

**解析**：本题是论述题热点。答题时需先分别解释两个概念，再阐述四重关系。注意结合当前图书馆实际，说明为什么两者必须并重。

</details>

### 题目5（论述题）

**论述信息资源建设从"藏书建设"到"信息资源建设"的发展演变及其驱动因素。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

**一、三阶段演变**

1. **藏书建设（20世纪50年代）**
   - 核心：图书馆根据性质、任务和读者需求，有计划地选择、收集、组织、保管、复选、剔除文献
   - 特点：以纸质图书为中心，强调"收藏"和"保管"

2. **文献资源建设（20世纪80年代）**
   - 核心：系统地规划、选择、收集、组织管理文献资源，建立具有特定功能的藏书体系
   - 特点：从单一图书扩展到各类文献，强调"系统"和"体系"

3. **信息资源建设（20世纪90年代至今）**
   - 核心：有计划地规划、选择、收集、组织、管理各种信息资源，建立具有特定功能的信息资源体系
   - 特点：涵盖多种媒介（纸质+数字），强调"共建共享"和"利用"

**二、驱动因素**

1. **技术驱动**：信息技术发展，数字资源涌现
2. **需求驱动**：用户信息需求多样化、个性化
3. **资源驱动**：信息数量激增，单一馆无法穷尽收藏
4. **理念驱动**：从"以藏为主"到"以用为主"，从"拥有"到"拥有+存取"
5. **政策驱动**：国家信息化发展战略推动

**解析**：本题是综合性论述题，需清晰呈现三阶段的时间、概念、特征，并从多角度分析演变原因。答题时注意逻辑递进，体现"范围扩大→载体扩展→理念升级"的内在脉络。

</details>
