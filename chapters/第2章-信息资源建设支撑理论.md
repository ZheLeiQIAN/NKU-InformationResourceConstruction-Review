## 【核心概念】

### ★ 必考概念

| 概念 | 定义 | 关键要点 |
|------|------|----------|
| **系统（System）** | 由两个或两个以上事物按照一定客观规律，相互联系、相互作用结合而成的具有特定功能的统一体 | 要素+联系+功能+统一体 |
| **信息资源保障体系** | 基于系统整体性原则，建立的有明确目标、与社会环境相适应的信息资源系统 | 整体性原则的应用 |
| **共建共享** | 一个地区、系统、国家乃至全球的图书馆之间建立广泛合作关系，科学规划、分工协作、共同建设、相互提供利用 | 联系性原则的核心应用 |
| **特色化建设** | 图书馆信息资源具有独特风格，与其他图书馆不同的特点 | 理念：人无我有、人有我多、人多我精 |
| **信息生命周期** | 信息从生成到失效的全过程：生成→采集→存储/组织/加工→利用→维护→失效/剔除 | 信息资源建设就是完成一个生命周期的过程 |
| **布拉德福定律** | 将期刊按载文量分为三区，每区文献数约为总数的1/3，期刊数成等比数列（公比≈5） | P1:P2:P3 = 1:n:n² |
| **核心区/相关区/相邻区** | 核心区：载文量最高的少数核心期刊；相关区：载文量中等的较多期刊；相邻区：载文量最低而数量最多的期刊 | 三区定义是常考点 |
| **普赖斯曲线** | 文献增长与时间成指数函数关系：F(t) = ae^(bt) | 指数增长模型 |
| **文献老化** | 文献内容的老化和文献载体的老化 | 内容老化是研究重点 |
| **半衰期（Half-life）** | 某学科现时尚在利用的全部文献中，较新的一半是在多长时间内发表或出版的 | 各学科半衰期不同 |
| **引文年代分析法** | 统计论文所附引文的发表时间和被引用时间，分析文献老化规律 | 文献老化研究方法之一 |
| **价值论选书观** | 杜威、海恩斯主张选择科学或艺术价值高的出版物，以文献内容价值为选择依据 | 片面性：不考虑读者需求 |
| **需要论选书观** | 普尔·克特强调藏书补充的实用性，以用户需要为依据 | 片面性：忽视社会阅读导向 |
| **使用概率论** | 阮冈纳赞提出入藏每本书是否被利用是随机事件，概率越大表示被利用可能性越大 | 引入数学方法，客观但操作难 |
| **成本-收益理论** | 通过调控手段协调信息资源在时间、空间、部门、数量上的分布，使有限资源产生最大效益 | 经济学命题 |

### ☆ 重要概念

| 概念 | 定义 | 关键要点 |
|------|------|----------|
| **系统的整体性原则** | 信息资源保障体系的方法论基础，必须建立保障体系、明确目标、适应环境 | 三个"必须" |
| **系统的联系性原则** | 为信息资源体系结构研究提供理论依据，要求合理联系宏观规划、改善微观结构、有机布局资源 | 共建共享的理论基础 |
| **系统的有序性原则** | 为信息资源组织提供理论指导，要求进行揭示加工整序、有序化布局、建立检索系统 | 组织管理的理论依据 |
| **系统的动态性** | 要求信息资源建设适应社会信息需求变化，关注时间趋势、用户需求、外部环境 | 持续发展的理论依据 |
| **稀缺型特色资源** | 图书馆独具特色的部分馆藏 | 特色化建设层次之一 |
| **完备型特色资源** | 对某一学科或某一类型文献具有系统性收藏的馆藏 | 特色化建设层次之一 |
| **特色化建设六类型** | 专题特色、学科特色、地方特色、时代特色、民族特色、文献类型特色 | 分类记忆 |
| **综合计分法** | 麦高文从文献知识价值和社会需求两方面综合评价 | 辩证但难实行 |
| **社会调查选择论** | 维拉德依据对读者的社会、心理研究结果进行选书 | 按年龄、教育、职业分组 |
| **阅读吸引力** | 戴利2014年提出，包括图书相关吸引力和读者驱动吸引力 | 双向吸引力模型 |
| **阅读时态模型** | 戴利提出：阅读前（预期效果）→阅读时（吸引力元素）→阅读后（阅读作用） | 三阶段时态 |
| **会计成本** | 实际发生的成本 | 成本类型之一 |
| **机会成本** | 选择某种东西而放弃的其他东西的成本 | 成本类型之一 |

## 【重点公式/算法】

### 1. 布拉德福定律公式 ★★★

```
P1 : P2 : P3 = 1 : n : n²

其中：
- P1 = 核心区期刊数量
- P2 = 相关区期刊数量  
- P3 = 相邻区期刊数量
- n ≈ 5（公比近似为5）
```

**含义**：将某学科期刊按载文量降序排列，分为三个区，每区载文量约为该学科总文献数的1/3，但三个区的期刊数量之比约为 1 : 5 : 25。

**示例**：某学科共有900篇论文分布在100种期刊上
- 第一区（核心区）：约300篇论文，分布在约5种期刊上
- 第二区（相关区）：约300篇论文，分布在约25种期刊上
- 第三区（相邻区）：约300篇论文，分布在约70种期刊上

**对信息资源建设的意义**：
1. 确定本学科核心期刊，以有限经费获得高质量、使用率高的文献
2. 确定合理的藏书规模，规划馆藏文献布局

### 2. 普赖斯曲线（文献增长公式）★★

```
F(t) = a × e^(bt)

其中：
- F(t) = 时刻t的文献累积量
- a = 统计初始时刻的文献量
- e = 2.718（自然对数底）
- b = 持续增长率
- t = 时间
```

**含义**：文献增长与时间成指数函数关系。

**对信息资源建设的意义**：
1. 在一定程度上正确反映了文献的实际增长情况
2. 为文献信息资源建设提供了丰富的信息源，但也产生了"信息污染"

**局限性**：
1. 适合于一定历史时期和学科，不能推而广之
2. 不同文献类型、学科在不同历史时期有不同的增长规律

### 4. 特色化建设目标公式（理念层面）

```
特色化建设理念 = 人无我有 + 人有我多 + 人多我精

数量目标：尽可能完整系统地收集、入藏的完备程度
质量目标：有一定深度、反映前沿、结构合理
类型目标：虚拟资源、文献资源、重点藏书与核心期刊
```

### 5. 信息生命周期流程

```
生成 → 采集 → 存储/组织/加工 → 利用 → 维护 → 失效/剔除
 │       │            │           │       │         │
 │       │            │           │       │         └─ 剔除决策
 │       │            │           │       └─ 内容更新、载体维护
 │       │            │           └─ 信息价值实现
 │       │            └─ 系统化、增值
 │       └─ 集中化/专项/随机
 └─ 原始信息分散生成；加工后生成新信息
```

**核心结论**：信息资源建设就是完成一个信息生命周期的过程。

## 【方法比较】

### 1. 系统理论四大原则比较

| 原则 | 核心内涵 | 在信息资源建设中的应用 | 实践要求 |
|------|----------|------------------------|----------|
| **整体性原则** | 系统是有机整体，整体功能大于部分之和 | 建立信息资源保障体系 | 必须有明确目标、适应社会环境 |
| **联系性原则** | 系统要素相互联系、相互作用 | 共建共享、体系结构研究 | 统筹安排、专门化系统、馆际互借 |
| **有序性原则** | 系统结构有序、层次分明 | 信息资源组织 | 揭示加工整序、有序化布局、检索系统 |
| **动态性原则** | 系统随时间变化发展 | 适应社会信息需求变化 | 关注趋势、用户需求、外部环境 |

**记忆口诀**：整体建体系，联系促共享，有序管组织，动态应变化。

### 2. 五大选书理论比较 ★★★

| 选书理论 | 代表人物 | 核心观点 | 优点 | 片面性/局限 |
|----------|----------|----------|------|-------------|
| **价值论** | 杜威、海恩斯 | 选择科学或艺术价值高的出版物 | 重视文献质量 | 只重价值，不考虑读者实际需求 |
| **需要论** | 普尔·克特 | 以用户需要为依据，强调实用性 | 重视用户需求 | 忽视图书馆社会阅读导向作用 |
| **综合计分法** | 麦高文 | 从知识价值和社会需求两方面综合评价 | 具有辩证观点 | 实践中难以实行 |
| **社会调查选择论** | 维拉德 | 依据对读者的社会、心理研究结果选书 | 科学分组分析 | 操作复杂，时效性受限 |

**考试提示**：常考"片面性"的对应关系——价值论忽视需求，需要论忽视导向。

### 题目1（名词解释）

**请解释"布拉德福定律"及其对信息资源建设的意义。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

**布拉德福定律**（文献分布定律）是英国文献学家布拉德福通过对应用地球物理学和润滑领域490种期刊、1727篇论文的统计研究发现的文献分布规律。

**核心内容**：
1. 某一学科文献在期刊上载文量的多少，随该期刊与学科关系疏密程度增减变化
2. 按专业文献载文量多少，可将期刊划分为三个区域：
   - **核心区（P1）**：载文量最高的少数核心期刊
   - **相关区（P2）**：载文量中等的数量较多的期刊
   - **相邻区（P3）**：载文量最低而数量最多的期刊
3. 每区中期刊登载某一学科文献数量约为该学科所发表文献总数的1/3
4. 三个区域的期刊数量之比成几何级数分布：P1 : P2 : P3 = 1 : n : n²（n约为5）

**对信息资源建设的意义**：
1. 确定本学科核心期刊，以有限经费获得高质量、使用率高的文献
2. 确定合理的藏书规模，规划馆藏文献布局

**解析**：本题是必考名词解释。答题时需写出发现背景、核心内容（三区定义+公式）、建设意义。注意公式 P1:P2:P3 = 1:n:n² 中 n≈5 是常考数字。

</details>

### 题目2（计算题）

**某学科共有1800篇论文分布在期刊上，根据布拉德福定律，若核心区有10种期刊，请计算相关区和相邻区的期刊数量，并验证三区论文分布。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

已知：总论文数 = 1800篇，核心区期刊数 P1 = 10种，n ≈ 5

**计算期刊数量**：
- 核心区期刊数 P1 = 10种
- 相关区期刊数 P2 = P1 × n = 10 × 5 = 50种
- 相邻区期刊数 P3 = P1 × n² = 10 × 25 = 250种
- 总期刊数 = 10 + 50 + 250 = 310种

**验证论文分布**：
- 每区论文数 ≈ 1800 ÷ 3 = 600篇
- 核心区：10种期刊刊载约600篇论文，平均每刊60篇
- 相关区：50种期刊刊载约600篇论文，平均每刊12篇
- 相邻区：250种期刊刊载约600篇论文，平均每刊2.4篇

**结论**：符合布拉德福定律"每区论文数约占总数的1/3，期刊数成等比数列"的规律。

**解析**：本题是布拉德福定律的典型计算题。解题关键是记住公式 P1:P2:P3 = 1:n:n²，已知P1和n，可求出P2和P3。注意验证时说明"每区论文数约为总数的1/3"。

</details>

### 题目4（简答题）

**简述文献老化的概念、研究方法和影响因素。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

**一、文献老化的概念**

文献老化包括两个方面：
1. **文献内容的老化**：文献中所含信息逐渐失去价值，被新文献替代
2. **文献载体的老化**：文献物理载体的损坏、变质

**半衰期**：某学科现时尚在利用的全部文献中，较新的一半是在多长时间内发表或出版的。

**二、文献老化的研究方法**

1. **引文年代分析法**：统计论文所附引文的发表时间和被引用时间
2. **文献流通统计法**：通过借阅流通数据统计分析利用规律和老化规律
3. **综合统计分析法**：以引文数据、流通数据、馆内使用数据等为基础综合分析

**三、文献老化的影响因素**

1. **学科的差异**：不同学科半衰期不同（如地理学16.1年，生物医学3.0年）
2. **学科发展阶段的差异**：前科学→常规科学→反常与危机→科学革命
3. **文献的种类和性质**：产品样本、期刊、技术论文、学位论文、科技报告、图书半衰期递增
4. **衡量标准**：引用标准 vs 借阅标准，结果可能不同

**四、对信息资源建设的意义**

1. 为评价和选择文献信息资源提供理论依据
2. 为优化馆藏文献资源的结构提供理论依据
3. 为制定科学合理的文献工作原则提供理论依据

**解析**：本题是综合性简答题。答题时需涵盖概念、方法、影响因素、意义四个层面。注意半衰期数据可举1-2个例子，不必全部背诵。

### 题目5（论述题）

**论述系统理论的整体性、联系性、有序性、动态性四大原则在信息资源建设中的应用。**

<details>
<summary>点击查看答案与解析</summary>

**答案：**

**一、整体性原则**

- **核心内涵**：系统是有机整体，整体功能大于部分之和
- **应用**：是信息资源保障体系的方法论基础
  - 必须建立信息资源保障体系
  - 信息资源保障体系必须有明确的系统目标
  - 信息资源保障体系的运行必须与社会环境相适应
- **实践意义**：单个图书馆无法满足所有信息需求，必须从整体出发构建保障体系

**二、联系性原则**

- **核心内涵**：系统要素相互联系、相互作用
- **应用**：为信息资源体系结构研究提供理论依据
  - **共建共享**：一个地区、系统、国家乃至全球的图书馆之间建立广泛合作关系，科学规划、分工协作、共同建设、相互提供利用
  - 要求：从整体目标出发统筹安排、各图书馆建立有重点有特色的专门化系统、通过馆际互借和文献传递实现资源共享
  - 合理联系宏观规划、改善微观资源结构、资源布局有机联系
- **实践意义**：打破"各自为政"的壁垒，实现资源优化配置

**三、有序性原则**

- **核心内涵**：系统结构有序、层次分明
- **应用**：为信息资源组织提供理论指导
  - 对信息资源进行揭示、加工和整序
  - 对各类信息资源有序化布局排列
  - 建立完善的信息资源检索系统
- **实践意义**：无序的信息无法被有效利用，必须通过组织实现有序化

**四、动态性原则**

- **核心内涵**：系统随时间变化发展
- **应用**：要求信息资源建设必须适应社会信息需求的变化
  - 研究信息资源在时间上发展变化的趋势和规律
  - 随时关注用户信息需求的变化
  - 研究信息资源建设面临的外部环境变化
- **实践意义**：信息资源建设不是一劳永逸的，必须持续更新、动态调整

**总结**：四大原则相互关联、层层递进——整体性确立目标，联系性构建网络，有序性组织内容，动态性保障发展，共同构成信息资源建设的系统方法论。

**解析**：本题是论述题热点。答题时需分别阐述四大原则的内涵、应用、实践意义，最后进行总结。注意联系性原则中的"共建共享"是高频考点，需重点展开。

### 易错点提醒

| 易错点 | 正确内容 |
|--------|----------|
| 布拉德福公比是3 | 公比近似为 **5** |
| 普赖斯曲线是线性增长 | 是 **指数增长** F(t)=ae^(bt) |
| 半衰期所有学科一样 | 不同学科半衰期 **差异很大** |
| 价值论重视读者需求 | 价值论 **忽视** 读者需求，需要论才重视 |
| 需要论重视文献质量 | 需要论 **忽视** 社会阅读导向 |
| 信息生命周期只有5个阶段 | 有 **6个** 阶段（生成→采集→存储/组织/加工→利用→维护→失效/剔除） |
| 特色化建设只有馆藏特色 | 还包括 **特色数据库建设** |
